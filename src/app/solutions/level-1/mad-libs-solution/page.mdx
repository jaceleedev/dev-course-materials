# 🎭 Mad Libs 게임 (답안)

## 🚀 기본 기능 답안

```python
# 🎭 Mad Libs 게임
print("🎭 Mad Libs 게임에 오신 것을 환영합니다!")
print("재미있는 이야기를 만들어보세요!")
print("="*50)

# 단어 수집
name = input("이름: ")
adjective1 = input("형용사 1: ")
noun1 = input("명사 1: ")
verb1 = input("동사 1: ")
adjective2 = input("형용사 2: ")
noun2 = input("명사 2: ")
number = input("숫자: ")

# 이야기 생성
print("\n" + "="*50)
print("🎪 완성된 이야기")
print("="*50)

story = f"""
옛날 옛적에 {adjective1}한 {name}이(가) 살고 있었습니다.
{name}은(는) 매일 {noun1}을(를) {verb1}하는 것을 좋아했습니다.
어느 날, {adjective2}한 {noun2}이(가) 나타나서 말했습니다:
"{name}님, {number}개의 소원을 들어드리겠습니다!"
그리하여 {name}은(는) 행복하게 살았답니다.
"""

print(story)
print("="*50)
print("🎉 이야기가 완성되었습니다!")
```

## 🎨 확장 버전 (더 많은 단어 사용)

```python
# 🎭 Mad Libs 게임 (확장 버전)
print("🎭 Mad Libs 게임 - 확장 버전")
print("="*40)

# 더 많은 단어 수집
hero_name = input("주인공 이름: ")
adjective_appearance = input("외모를 나타내는 형용사: ")
place = input("장소: ")
object1 = input("물건 1: ")
action1 = input("행동 1: ")
adjective_feeling = input("감정을 나타내는 형용사: ")
animal = input("동물: ")
food = input("음식: ")
number = input("숫자: ")

# 확장된 이야기 생성
print("\n" + "="*40)
print("📖 모험 이야기")
print("="*40)

extended_story = f"""
{adjective_appearance}한 {hero_name}은(는) {place}에서 살고 있었습니다.
어느 날 {hero_name}은(는) 신비한 {object1}을(를) 발견했습니다.
{object1}을(를) {action1}하자, 갑자기 {adjective_feeling}한 기분이 들었습니다.

그때 말하는 {animal}이(가) 나타나서 말했습니다:
"축하합니다! 당신은 {number}개의 소원을 빌 수 있어요!"

{hero_name}은(는) 첫 번째 소원으로 맛있는 {food}을(를) 달라고 했습니다.
그리고 모든 사람들이 행복하기를 바랐습니다.

마지막 소원은... 더 많은 모험을 떠나는 것이었습니다!
"""

print(extended_story)
print("="*40)
print("🌟 모험이 계속됩니다!")
```

## 💡 핵심 포인트

1. **문자열 조작**: f-string을 활용한 변수 삽입
2. **여러 줄 문자열**: `"""` 사용으로 긴 이야기 작성
3. **사용자 입력**: `input()` 함수로 다양한 단어 수집
4. **창의적 구성**: 시작-중간-끝 구조로 완성도 있는 이야기 작성
