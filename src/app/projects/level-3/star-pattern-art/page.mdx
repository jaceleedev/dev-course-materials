# ⭐ 별찍기 아트

## 📋 프로젝트 개요

중첩 반복문의 마법으로 아름다운 별 패턴과 기하학적 도형을 만들어보세요. 단순한 삼각형부터 복잡한 다이아몬드까지, 코드로 예술 작품을 창조하는 특별한 경험을 해봅시다!

## 🎯 학습 목표

- 중첩 for문을 활용한 2차원 패턴 구현
- 수학적 사고를 통한 패턴 분석과 구현
- 조건문과 반복문의 조합 활용
- 사용자 정의 패턴 생성 시스템 구축

## 🚀 기본 기능

### 핵심 요구사항

1. **기본 삼각형 패턴들**

   - 우직각삼각형 (왼쪽 정렬)
   - 우직각삼각형 (오른쪽 정렬)
   - 이등변삼각형 (중앙 정렬)

2. **사용자 입력**

   - 원하는 패턴 크기 선택 (5~20줄)
   - 패턴 종류 선택

3. **깔끔한 출력**
   - 정확한 정렬과 간격
   - 패턴별 설명과 이름 표시

## 🎨 추가 기능 1: 고급 패턴

### 요구사항

- **다이아몬드**: 상하 대칭 다이아몬드 모양
- **역삼각형**: 아래로 갈수록 작아지는 삼각형
- **속이 빈 삼각형**: 테두리만 있는 삼각형
- **피라미드**: 계단식 피라미드

### 복잡한 수학적 패턴

- 각 패턴의 행별 별 개수 계산
- 정확한 공백 계산 공식
- 대칭성과 규칙성 구현

## 💡 추가 기능 2: 커스터마이징 시스템

### 요구사항

- **사용자 정의 문자**: 별 대신 다른 문자 사용
- **크기 조절**: 1~30줄까지 자유롭게 조절
- **색상 패턴**: 다양한 문자 조합으로 색상 효과
- **패턴 조합**: 여러 패턴을 한 번에 출력

### 개인화 옵션

- 이름이나 글자를 패턴으로 만들기
- 무지개 색상 효과 (다양한 문자 사용)

## 🔥 추가 기능 3: 점진적 패턴 시스템

### 요구사항

- **단계별 패턴**: 한 줄씩 차례대로 그려지는 패턴
- **다양한 문자 패턴**: 별, 숫자, 알파벳을 사용한 패턴
- **패턴 변환**: 같은 모양을 다른 문자로 변환
- **크기 조절**: 사용자가 원하는 크기로 패턴 생성

### 인터랙티브 기능

- **단계별 확인**: 각 줄마다 사용자 확인 후 다음 줄 출력
- **문자 선택**: 사용자가 원하는 문자로 패턴 생성
- **패턴 비교**: 여러 패턴을 나란히 비교

### 학습 효과

- **패턴 이해**: 각 줄의 규칙을 단계별로 이해
- **수학적 사고**: 줄 번호와 문자 개수의 관계 파악

#### 5. 속이 빈 삼각형

```python
# 속이 빈 삼각형
height = int(input("삼각형 높이를 입력하세요 (5-15): "))

print("🔺 속이 빈 삼각형")
print("-" * 20)

for i in range(1, height + 1):
    # 앞쪽 공백
    for j in range(height - i):
        print(" ", end="")

    # 각 줄별 처리
    for j in range(2 * i - 1):
        if i == 1 or i == height:  # 첫 줄 또는 마지막 줄
            print("*", end="")
        else:  # 중간 줄들
            if j == 0 or j == 2 * i - 2:  # 양끝만
                print("*", end="")
            else:  # 가운데는 공백
                print(" ", end="")

    print()

# 실행 예시: height=5
#     *
#    * *
#   *   *
#  *     *
# *********
```

#### 11. 점진적 패턴 시스템

```python
# 점진적 패턴 시스템 (기본 변수와 반복문만 사용)
print("🎨 점진적 패턴 시스템")
print("="*30)

# 단계별 삼각형 그리기
print("\n⭐ 단계별 삼각형 그리기")
height = int(input("삼각형 높이 (5-8): "))

for i in range(1, height + 1):
    print(f"\n📏 {i}번째 줄 그리기:")

    # 현재 줄까지의 삼각형 보여주기
    for j in range(1, i + 1):
        # 앞쪽 공백
        for k in range(height - j):
            print(" ", end="")

        # 별 출력
        for k in range(2 * j - 1):
            print("*", end="")

        print()

    # 진행 상태 표시
    progress = (i * 100) // height
    print(f"진행률: {progress}%")

    # 다음 줄 확인
    if i < height:
        continue_draw = input("다음 줄을 그릴까요? (y/n): ")
        if continue_draw.lower() != 'y':
            break

print("\n🎉 패턴 완성!")

# 실행 예시: height=4
# 📏 1번째 줄 그리기:
#    *
# 진행률: 25%
#
# 📏 2번째 줄 그리기:
#    *
#   ***
# 진행률: 50%
#
# 📏 3번째 줄 그리기:
#    *
#   ***
#  *****
# 진행률: 75%
#
# 📏 4번째 줄 그리기:
#    *
#   ***
#  *****
# *******
# 진행률: 100%

# 패턴 변환 시스템
print("\n🔄 패턴 변환 시스템")
height = int(input("패턴 높이 (5-8): "))

# 다양한 문자들
characters = ['*', '#', '@', '$', '%']
char_names = ['별', '샵', '골뱅이', '달러', '퍼센트']

for idx in range(len(characters)):
    char = characters[idx]
    name = char_names[idx]

    print(f"\n🎨 {name}({char}) 패턴:")

    # 삼각형 그리기
    for i in range(1, height + 1):
        for j in range(height - i):
            print(" ", end="")
        for j in range(2 * i - 1):
            print(char, end="")
        print()

    # 다음 패턴 확인
    if idx < len(characters) - 1:
        next_pattern = input("다음 패턴을 보시겠습니까? (y/n): ")
        if next_pattern.lower() != 'y':
            break

print("\n🌟 패턴 변환 완료!")

# 실행 예시: height=3
# 🎨 별(*) 패턴:
#   *
#  ***
# *****
#
# 🎨 샵(#) 패턴:
#   #
#  ###
# #####
#
# 🎨 골뱅이(@) 패턴:
#   @
#  @@@
# @@@@@
```

### 중첩 반복문 활용 포인트

- **바깥쪽 for문**: 줄 수 제어 (세로 방향)
- **안쪽 for문**: 각 줄의 문자 출력 (가로 방향)
- **공백 출력**: 정렬을 위한 공백 개수 계산
- **별 출력**: 각 줄의 별 개수 계산

### 패턴 분석 방법

1. **줄별 분석**: 각 줄에 몇 개의 공백과 별이 필요한가?
2. **규칙 찾기**: 줄 번호에 따른 공백과 별의 개수 공식
3. **대칭성 활용**: 다이아몬드는 위아래 대칭
4. **수학적 접근**: 홀수, 짝수 개념 활용

- **시각적 효과**: 다양한 문자와 색상 활용
- **사용자 참여**: 높이와 문자를 직접 선택
- **단계별 구현**: 간단한 패턴부터 복잡한 패턴으로
- **오류 처리**: 잘못된 입력에 대한 적절한 처리

### 도전 과제

1. **무지개 패턴**: 각 줄마다 다른 문자 사용
2. **회전 패턴**: 삼각형을 다른 방향으로 회전
3. **복합 패턴**: 여러 패턴을 조합한 복잡한 모양
