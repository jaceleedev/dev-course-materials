# 🔢 구구단 마스터

## 📋 프로젝트 개요

반복문을 활용하여 구구단을 자유자재로 다루는 완전한 마스터 프로그램을 만들어보세요. 단순한 출력부터 재미있는 퀴즈 게임까지, 구구단의 모든 것을 담은 프로젝트입니다!

## 🎯 학습 목표

- for문과 while문을 활용한 반복 처리
- 중첩 반복문으로 2차원 패턴 구현
- 사용자 상호작용과 반복문의 조합
- 시간 측정과 점수 시스템 구현

## 🚀 기본 기능

### 핵심 요구사항

1. **원하는 단 출력**

   - 사용자가 원하는 구구단 단수 입력
   - 해당 단의 구구단 완전 출력 (1~9)
   - 깔끔한 형식으로 정렬

2. **입력 검증**

   - 1~9 범위 외 입력에 대한 처리
   - 재입력 요청 시스템

3. **기본 출력 형식**
   - "3 × 4 = 12" 형태로 출력
   - 보기 좋은 정렬과 구분선

## 🎨 추가 기능 1: 구구단 퀴즈 게임

### 요구사항

- 랜덤한 구구단 문제 출제 (2단~9단)
- 사용자 답 입력 및 정답 확인
- 연속 문제 풀이 (5문제 또는 틀릴 때까지)
- 정답률 계산 및 표시

### 게임 진행 방식

- "3 × 7 = ?" 형태로 문제 출제
- 정답/오답 즉시 피드백
- 최종 점수 및 평가 제공

## 💡 추가 기능 2: 시간 측정 시스템

### 요구사항

- 문제 풀이 시간 측정
- 시간 제한 설정 (예: 10초)
- 빠른 답변 보너스 점수
- 시간 초과 시 자동 다음 문제

### 점수 계산 방식

- 기본 점수: 정답 시 10점
- 시간 보너스: 5초 이내 +5점, 3초 이내 +10점
- 연속 정답 보너스: 연속 3개 이상 +5점씩

## 🔥 추가 기능 3: 레벨별 도전 시스템

### 요구사항

- **레벨 1**: 기본 (5문제, 10초 제한)
- **레벨 2**: 도전 (10문제, 8초 제한)
- **레벨 3**: 전문가 (15문제, 6초 제한)
- **레벨 4**: 마스터 (20문제, 4초 제한)

### 승급 조건

- 각 레벨에서 80% 이상 정답률 달성
- 평균 응답 시간 기준 달성
- 연속 정답 기록 달성

### 특별 보상

- **완벽 클리어**: 모든 문제 정답 시 보너스 점수
- **스피드 보너스**: 제한 시간의 절반 이내 완료
- **마스터 인증**: 모든 레벨 클리어 시 특별 칭호

#### 3. 구구단 퀴즈 게임

```python
import random

# 퀴즈 게임 설정
score = 0
total_questions = 5

print("🎮 구구단 퀴즈 게임을 시작합니다!")

for question_num in range(1, total_questions + 1):
    # 랜덤 문제 생성
    num1 = random.randint(2, 9)
    num2 = random.randint(1, 9)
    correct_answer = num1 * num2

    print(f"\n📝 문제 {question_num}: {num1} × {num2} = ?")
    user_answer = int(input("답: "))

    if user_answer == correct_answer:
        print("🎉 정답입니다!")
        score += 1
    else:
        print(f"❌ 틀렸습니다. 정답은 {correct_answer}입니다.")

print(f"\n📊 최종 점수: {score}/{total_questions}")

# 성적 평가
if score == total_questions:
    print("🏆 완벽해요! 구구단 마스터!")
elif score >= total_questions * 0.8:
    print("🌟 우수해요!")
elif score >= total_questions * 0.6:
    print("👍 잘했어요!")
else:
    print("📚 조금 더 연습해보세요!")
```

#### 4. 레벨별 도전 시스템

```python
# 레벨별 설정 (기본 변수 사용)
current_level = 1
print("🎯 구구단 마스터 도전!")

while current_level <= 4:
    # 레벨별 문제 수 설정
    if current_level == 1:
        level_name = "기본"
        question_count = 5
        pass_score = 4  # 80% = 5문제 중 4문제
    elif current_level == 2:
        level_name = "도전"
        question_count = 8
        pass_score = 6  # 75% = 8문제 중 6문제
    elif current_level == 3:
        level_name = "전문가"
        question_count = 10
        pass_score = 8  # 80% = 10문제 중 8문제
    else:  # level 4
        level_name = "마스터"
        question_count = 15
        pass_score = 12  # 80% = 15문제 중 12문제

    print(f"\n🔥 레벨 {current_level}: {level_name}")
    print(f"📝 문제 수: {question_count}개")
    print(f"🎯 통과 기준: {pass_score}개 이상 정답")

    input("준비되면 엔터를 누르세요...")

    # 레벨 도전 시작
    score = 0

    for i in range(question_count):
        # 문제 생성 (2~9단, 1~9 곱하기)
        num1 = (i % 8) + 2  # 2~9단 순서대로
        num2 = (i % 9) + 1  # 1~9 순서대로
        answer = num1 * num2

        print(f"\n📝 문제 {i+1}: {num1} × {num2} = ?")
        user_answer = int(input("답: "))

        # 정답 확인
        if user_answer == answer:
            print("🎉 정답!")
            score += 1
        else:
            print(f"❌ 틀렸습니다. 정답: {answer}")

    # 레벨 결과 판정
    print(f"\n📊 레벨 {current_level} 결과:")
    print(f"🎯 정답: {score}/{question_count}")
    print(f"📈 정답률: {score * 100 // question_count}%")

    # 승급 조건 확인
    if score >= pass_score:
        print(f"🎉 레벨 {current_level} 클리어!")

        # 특별 보너스
        if score == question_count:
            print("💎 완벽 클리어 보너스!")

        current_level += 1

        if current_level <= 4:
            continue_game = input("\n다음 레벨에 도전하시겠습니까? (y/n): ")
            if continue_game.lower() != 'y':
                break
    else:
        print(f"😢 레벨 {current_level} 실패. {pass_score}개 이상 맞춰야 합니다!")
        retry = input("재도전하시겠습니까? (y/n): ")
        if retry.lower() != 'y':
            break

# 최종 결과
if current_level > 4:
    print("\n🏆 축하합니다! 구구단 마스터가 되었습니다!")
    print("🎖️ 모든 레벨을 정복한 진정한 마스터입니다!")
else:
    print(f"\n💪 레벨 {current_level-1}까지 도달했습니다!")
    print("🔥 다음에 더 높은 레벨에 도전해보세요!")
```

#### 9. 사용자 정의 범위 구구단

```python
# 사용자가 원하는 범위의 구구단 출력
print("📚 원하는 범위의 구구단을 출력해보세요!")

start_dan = int(input("시작 단 (2-9): "))
end_dan = int(input("끝 단 (2-9): "))

if start_dan > end_dan:
    print("❌ 시작 단이 끝 단보다 클 수 없습니다!")
elif start_dan < 2 or end_dan > 9:
    print("❌ 2~9 사이의 숫자를 입력하세요!")
else:
    print(f"📊 {start_dan}단부터 {end_dan}단까지")
    print("="*50)

    for dan in range(start_dan, end_dan + 1):
        print(f"\n📖 {dan}단")
        print("-"*15)

        for i in range(1, 10):
            result = dan * i
            print(f"{dan} × {i} = {result}")
```

### 반복문 활용 포인트

- **for문**: 정해진 횟수만큼 반복 (1~9까지, 2~9단까지)
- **while문**: 조건이 참인 동안 반복 (메뉴 시스템, 올바른 입력까지)
- **중첩 반복문**: 2차원 구조 (단 × 곱하는 수)
- **range()**: 숫자 범위 생성 (1~9, 2~9)

### 게임 설계 팁

- **난이도 조절**: 문제 수와 시간 제한으로 난이도 조정
- **피드백 시스템**: 즉각적인 정답/오답 피드백
- **보상 시스템**: 연속 정답과 빠른 답변에 보너스
- **사용자 경험**: 명확한 메뉴와 안내 메시지
