# 🎲 주사위 도박 게임

## 📋 프로젝트 개요

운과 전략이 만나는 스릴 넘치는 주사위 카지노를 만들어보세요! 포인트 베팅부터 연속 승리 보너스까지, 완전한 도박 게임 시스템을 구현하며 확률과 게임 이론을 체험해봅시다.

**⚠️ 교육 목적**: 이 프로젝트는 프로그래밍 학습을 위한 것으로, 실제 도박을 권장하지 않습니다.

## 🎯 학습 목표

- 게임 상태 관리와 복잡한 게임 로직 구현
- 확률 계산과 배당률 시스템 이해
- 사용자 자산 관리와 베팅 시스템 구현
- 반복문과 조건문을 활용한 종합적 프로그래밍

## 🚀 기본 기능

### 핵심 요구사항

1. **포인트 베팅 시스템**

   - 시작 포인트: 1000점
   - 베팅 금액 선택 (최소 10점, 최대 보유 포인트)
   - 승리 시 베팅 금액만큼 획득
   - 패배 시 베팅 금액 차감

2. **기본 주사위 게임**

   - 두 개의 주사위 합 맞추기
   - 예측 범위: 2~12
   - 정확히 맞추면 베팅 금액 × 5배
   - 틀리면 베팅 금액 잃기

3. **게임 진행**
   - 포인트가 0이 되면 게임 오버
   - 언제든지 게임 중단 가능
   - 현재 포인트 실시간 표시

## 🎨 추가 기능 1: 고급 베팅 시스템

### 요구사항

- **홀/짝 베팅**: 주사위 합이 홀수/짝수 맞추기 (1.8배)
- **범위 베팅**: 낮음(2-6), 중간(7-8), 높음(9-12) (각각 다른 배당)
- **연속 승리 보너스**: 3연승 이상 시 추가 포인트
- **럭키 세븐**: 합이 7이면 특별 보너스

### 배당률 시스템

- 정확한 숫자 맞추기: 5배 (확률 낮음)
- 홀/짝: 1.8배 (확률 높음)
- 범위 베팅: 2~3배 (중간 확률)

## 💡 추가 기능 2: 다양한 게임 모드

### 요구사항

- **더블 오어 낫싱**: 이기면 2배, 지면 모든 것을 잃음
- **잭팟 모드**: 낮은 확률, 높은 배당 (10~50배)
- **안전 모드**: 높은 확률, 낮은 배당 (1.2~1.5배)
- **토너먼트**: 제한 시간 내 최대 포인트 달성

### 특별 이벤트

- **황금 주사위**: 랜덤하게 등장하는 특별 라운드
- **보너스 타임**: 일정 조건 달성 시 보너스 게임

## 🔥 추가 기능 3: 종합 관리 시스템

### 요구사항

- **파산 방지**: 마지막 100포인트일 때 경고
- **대출 시스템**: 파산 시 500포인트 대출 (이자 포함)
- **업적 시스템**: 다양한 달성 목표와 보상
- **통계 및 순위**: 최고 점수, 연승 기록 등

## 💡 구현 힌트

### 필요한 파이썬 기능들

#### 1. 게임 기본 구조

```python
import random

# 게임 시작
points = 1000
print("🎲 주사위 카지노에 오신 것을 환영합니다!")

while points > 0:
    print(f"💰 현재 포인트: {points}점")

    # 게임 계속 여부 확인
    continue_game = input("게임을 계속하시겠습니까? (y/n): ")
    if continue_game.lower() != 'y':
        break

    # 베팅 금액 입력
    while True:
        bet = int(input(f"베팅 금액 (10~{points}): "))
        if 10 <= bet <= points:
            break
        else:
            print(f"❌ 10~{points} 사이의 금액을 베팅하세요!")

    # 예측 입력
    prediction = int(input("예측하는 합 (2-12): "))

    # 주사위 굴리기
    dice1 = random.randint(1, 6)
    dice2 = random.randint(1, 6)
    total = dice1 + dice2

    print(f"🎲 주사위 결과: [{dice1}] [{dice2}] = {total}")

    # 결과 판정
    if total == prediction:
        winnings = bet * 5
        points += winnings
        print(f"🎉 정답! {bet} × 5 = {winnings}포인트 획득!")
    else:
        points -= bet
        print(f"😢 틀렸습니다. {bet}포인트 잃었습니다.")

    # 파산 경고
    if points <= 100:
        print("⚠️ 포인트가 부족합니다! 신중하게 베팅하세요!")

print(f"🎮 게임 종료! 최종 포인트: {points}점")
```

```python

choice = input("홀수(odd) 또는 짝수(even): ").lower()

dice1 = random.randint(1, 6)
dice2 = random.randint(1, 6)
total = dice1 + dice2

print(f"🎲 주사위 결과: [{dice1}] [{dice2}] = {total}")

# 홀/짝 판정
is_odd = total % 2 == 1

if (choice == "odd" and is_odd) or (choice == "even" and not is_odd):
    winnings = int(bet * 1.8)  # 1.8배
    points += winnings
    print(f"🎉 정답! {bet} × 1.8 = {winnings}포인트 획득!")
else:
    points -= bet
    print(f"😢 틀렸습니다. {bet}포인트 잃었습니다.")
```

#### 3. 범위 베팅 시스템

```python
# 범위 베팅 게임
print("1. 낮음 (2-6) - 3배")
print("2. 중간 (7-8) - 4배")
print("3. 높음 (9-12) - 3배")

choice = int(input("선택 (1-3): "))

# 주사위 굴리기
dice1 = random.randint(1, 6)
dice2 = random.randint(1, 6)
total = dice1 + dice2

print(f"🎲 주사위 결과: [{dice1}] [{dice2}] = {total}")

# 범위 판정
if choice == 1 and 2 <= total <= 6:
    winnings = bet * 3
    points += winnings
    print(f"🎉 낮음 적중! {bet} × 3 = {winnings}포인트 획득!")
elif choice == 2 and 7 <= total <= 8:
    winnings = bet * 4
    points += winnings
    print(f"🎉 중간 적중! {bet} × 4 = {winnings}포인트 획득!")
elif choice == 3 and 9 <= total <= 12:
    winnings = bet * 3
    points += winnings
    print(f"🎉 높음 적중! {bet} × 3 = {winnings}포인트 획득!")
else:
    points -= bet
    print(f"😢 빗나갔습니다. {bet}포인트 잃었습니다.")
```

### 게임 설계 팁

- **확률 계산**: 각 베팅의 당첨 확률을 계산하여 적절한 배당률 설정

- **사용자 경험**: 명확한 안내 메시지와 결과 표시
- **파산 방지**: 마지막 포인트일 때 경고 시스템
