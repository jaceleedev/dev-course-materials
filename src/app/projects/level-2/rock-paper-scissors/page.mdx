# ✂️ 가위바위보 게임

## 📋 프로젝트 개요

컴퓨터와 대결하는 전통적인 가위바위보 게임을 만들어보세요. 사용자 입력과 조건문을 활용하여 모든 승부 상황을 처리하는 완전한 게임을 구현해봅시다!

## 🎯 학습 목표

- 복잡한 조건문을 이용한 게임 로직 구현
- 다양한 승부 상황을 체계적으로 처리
- 모든 경우의 수를 고려한 완전한 로직 설계

## 🚀 기본 기능

### 핵심 요구사항

1. **사용자 입력 처리**

   - 가위(1), 바위(2), 보(3) 중 선택
   - 잘못된 입력에 대한 처리

2. **컴퓨터 선택 생성**

   - 사용자 입력을 활용한 컴퓨터 선택
   - 예측 불가능한 결과 생성

3. **승부 판정**

   - 사용자 승리, 컴퓨터 승리, 무승부 판단
   - 승부 규칙 정확히 적용

4. **결과 출력**
   - 양쪽 선택과 최종 결과 표시
   - 승부 이유 설명

## 🎨 추가 기능 1: 점수 시스템

### 요구사항

기본 한 판 게임을 확장하여 점수를 기록하고 연속으로 게임을 진행합니다.

- **점수 기록**: 사용자와 컴퓨터의 승수 기록
- **연속 게임**: 계속 게임 할지 묻기 (y/n)
- **실시간 점수 표시**: 매 게임 후 누적 점수 표시
- **최종 결과**: 게임 종료 시 전체 결과 발표

### 게임 진행 방식

- 매 게임 후 점수 업데이트
- 사용자가 원할 때까지 계속 진행

## 💡 추가 기능 2: 3판 2승제

### 요구사항

점수 시스템을 확장하여 정식 3판 2승제를 구현합니다.

- **3판 진행**: 정확히 3판 게임 진행
- **각 판 결과 저장**: 1판, 2판, 3판의 결과 기록
- **최종 승자 결정**: 2승을 먼저 달성한 쪽이 승리
- **상세 기록**: 각 판의 선택과 결과 요약

### 토너먼트 시스템

- 각 판별 결과 표시
- 전체 게임 기록 보기
- 승부 통계 제공

## 🔥 추가 기능 3: 통계 및 고급 기능

### 요구사항

이전 모든 기능을 통합하여 고급 통계와 분석 기능을 추가합니다.

- **선택 통계**: 사용자와 컴퓨터의 선택 빈도 분석
- **승률 계산**: 전체 승률과 각 선택별 승률
- **연승/연패 기록**: 최고 연승과 현재 연승 기록
- **패턴 분석**: 사용자의 선택 패턴 간단 분석

### 고급 기능

- 게임 기록 저장 및 표시
- 추천 전략 제공
- 재미있는 통계 정보

computer_choice = (random_number % 3) + 1

### 게임 로직 설계

#### 1. 승부 판정 로직

```python
# 가위바위보 승부 규칙
# 가위(1) vs 보(3) → 가위 승리
# 바위(2) vs 가위(1) → 바위 승리
# 보(3) vs 바위(2) → 보 승리

def check_winner(user, computer):
    if user == computer:
        return "무승부"
    elif (user == 1 and computer == 3) or \
         (user == 2 and computer == 1) or \
         (user == 3 and computer == 2):
        return "사용자 승리"
    else:
        return "컴퓨터 승리"
```

#### 2. 컴퓨터 선택 생성 방법

```python
# 방법 1: 사용자 입력 숫자 활용
random_num = int(input("숫자 입력 (1-100): "))
computer_choice = (random_num % 3) + 1

# 방법 2: 문자열 길이 활용
user_name = input("이름을 입력하세요: ")
computer_choice = (len(user_name) % 3) + 1

# 방법 3: 계산식 활용
birth_year = int(input("출생 연도: "))
computer_choice = ((birth_year * 7) % 3) + 1
```

#### 3. 통계 시스템

```python
# 선택 통계 (가위, 바위, 보)
choice_counts = [0, 0, 0, 0]  # 인덱스 0은 사용 안 함

# 각 게임에서 선택 카운트
choice_counts[user_choice] += 1

# 승률 계산
win_rate = (user_wins * 100) // total_games if total_games > 0 else 0
```

### 조건문 활용 패턴

#### 1. 다중 조건 처리

```python
# 복잡한 승부 판정
if user_choice == computer_choice:
    result = "무승부"
elif user_choice == 1:  # 가위
    if computer_choice == 3:  # 보
        result = "승리"
    else:  # 바위
        result = "패배"
elif user_choice == 2:  # 바위
    if computer_choice == 1:  # 가위
        result = "승리"
    else:  # 보
        result = "패배"
else:  # 보
    if computer_choice == 2:  # 바위
        result = "승리"
    else:  # 가위
        result = "패배"
```

#### 2. 입력 검증

```python
# 유효한 입력 확인
if user_choice < 1 or user_choice > 3:
    print("잘못된 선택입니다!")
    continue

# 숫자 입력 확인
while True:
    try:
        choice = int(input("선택: "))
        if 1 <= choice <= 3:
            break
        else:
            print("1, 2, 3 중에서 선택하세요!")
    except ValueError:
        print("숫자를 입력하세요!")
```
