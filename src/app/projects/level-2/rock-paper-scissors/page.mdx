# ✂️ 가위바위보 게임

## 📋 프로젝트 개요

컴퓨터와 대결하는 전통적인 가위바위보 게임을 만들어보세요. 랜덤 선택과 조건문을 활용하여 모든 승부 상황을 처리하는 완전한 게임을 구현해봅시다!

## 🎯 학습 목표

- 복잡한 조건문을 이용한 게임 로직 구현
- 랜덤 모듈을 활용한 컴퓨터 AI 구현
- 모든 경우의 수를 고려한 완전한 로직 설계

## 🚀 기본 기능

### 핵심 요구사항

1. **사용자 입력 처리**

   - 가위(1), 바위(2), 보(3) 중 선택
   - 잘못된 입력에 대한 처리

2. **컴퓨터 랜덤 선택**

   - 가위, 바위, 보 중 랜덤하게 선택
   - 선택 결과 표시

3. **승부 판정**

   - 사용자 승리, 컴퓨터 승리, 무승부 판단
   - 승부 규칙 정확히 적용

4. **결과 출력**
   - 양쪽 선택과 최종 결과 표시

## 🎨 추가 기능 1: 점수 시스템

### 요구사항

- 여러 판을 연속으로 진행
- 사용자와 컴퓨터의 승수 기록
- 최종 점수 표시 및 우승자 발표

### 게임 진행 방식

- 계속 게임 할지 묻기 (y/n)
- 누적 점수 실시간 표시

## 💡 추가 기능 2: 3판 2승제

### 요구사항

- 정확히 3판 진행 (반복문 없이 3번의 개별 게임)
- 각 판의 결과 저장
- 3판 결과를 바탕으로 최종 승자 결정
- 상세한 게임 기록 표시

## 💡 구현 힌트

### 필요한 파이썬 기능들


#### 5. 더 직관적인 승부 판정

```python
# 승리 조건을 명확히 표시
if user_choice == computer_choice:
    print("🤝 무승부입니다!")
elif user_choice == 1 and computer_choice == 3:
    print("✂️ 가위가 📜 보를 자릅니다! 당신이 승리!")
elif user_choice == 2 and computer_choice == 1:
    print("🗿 바위가 ✂️ 가위를 부숩니다! 당신이 승리!")
elif user_choice == 3 and computer_choice == 2:
    print("📜 보가 🗿 바위를 감쌉니다! 당신이 승리!")
else:
    # 컴퓨터 승리 경우들
    if computer_choice == 1 and user_choice == 3:
        print("✂️ 가위가 📜 보를 자릅니다! 컴퓨터 승리!")
    elif computer_choice == 2 and user_choice == 1:
        print("🗿 바위가 ✂️ 가위를 부숩니다! 컴퓨터 승리!")
    elif computer_choice == 3 and user_choice == 2:
        print("📜 보가 🗿 바위를 감쌉니다! 컴퓨터 승리!")
```

#### 6. 점수 표시


### 게임 규칙 정리

- **가위** ✂️ : 보를 이김, 바위에게 짐
- **바위** 🗿 : 가위를 이김, 보에게 짐
- **보** 📜 : 바위를 이김, 가위에게 짐

### 재미있는 추가 요소

- 승부 결과에 따른 다양한 메시지
- 이모지를 활용한 시각적 효과
- 연승/연패 기록 표시
- 승률 계산 및 표시

## 🎯 도전 과제

1. **확장된 게임**: 가위바위보보 (보 2개) 추가
2. **AI 패턴**: 컴퓨터가 사용자 패턴을 분석 (간단한 형태)
3. **토너먼트**: 여러 명이 참가하는 토너먼트 시스템
4. **통계 시스템**: 선택 빈도, 승률 등 상세 통계
