# 📊 학점 계산기

## 📋 프로젝트 개요

입력받은 점수를 바탕으로 학점을 계산하고, 장학금 대상 여부를 판단하는 실용적인 프로그램을 만들어보세요. 조건문을 활용하여 복잡한 판단 로직을 구현해봅시다!

## 🎯 학습 목표

- 다중 조건문(if-elif-else)을 활용한 복잡한 판단 로직 구현
- 논리 연산자를 이용한 복합 조건 처리
- 실생활 문제를 프로그래밍으로 해결하는 경험

## 🚀 기본 기능

### 핵심 요구사항

1. **점수 입력 및 검증**

   - 0~100점 범위의 점수 입력받기
   - 유효하지 않은 점수에 대한 처리

2. **학점 계산**

   - A: 90점 이상
   - B: 80점 이상 90점 미만
   - C: 70점 이상 80점 미만
   - D: 60점 이상 70점 미만
   - F: 60점 미만

3. **결과 출력**
   - 입력받은 점수와 해당하는 학점 표시

## 🎨 추가 기능 1: 장학금 대상 판정

### 요구사항

- **성적 우수 장학금**: A학점 (90점 이상)
- **성적 향상 장학금**: B학점 이상이면서 이전 학기 점수가 현재보다 낮은 경우
- **근로 장학금**: C학점 이상이면서 근로학생인 경우

### 추가 입력사항

- 이전 학기 점수
- 근로학생 여부 (y/n)

## 💡 추가 기능 2: 다중 과목 학점 계산

### 요구사항

- 3과목의 점수를 각각 입력받기
- 각 과목별 학점 계산
- 전체 평균 점수와 평균 학점 계산

## 💡 구현 힌트

### 필요한 파이썬 기능들

#### 1. 점수 입력 및 검증

```python
score = int(input("점수를 입력하세요 (0-100): "))

if score < 0 or score > 100:
    print("잘못된 점수입니다. 0-100 사이의 값을 입력하세요.")
else:
    # 학점 계산 로직
```

#### 2. 다중 조건문으로 학점 계산

```python
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"
```

#### 3. 복합 조건으로 장학금 판정

```python
# 성적 우수 장학금
if grade == "A":
    print("성적 우수 장학금 대상입니다!")

# 성적 향상 장학금
previous_score = int(input("이전 학기 점수: "))
if grade in ["A", "B"] and score > previous_score:
    print("성적 향상 장학금 대상입니다!")

# 근로 장학금
is_worker = input("근로학생인가요? (y/n): ")
if score >= 70 and is_worker == "y":
    print("근로 장학금 대상입니다!")
```

#### 4. 학점을 점수로 변환 (GPA 계산용)

```python
if grade == "A":
    grade_point = 4.0
elif grade == "B":
    grade_point = 3.0
elif grade == "C":
    grade_point = 2.0
elif grade == "D":
    grade_point = 1.0
else:
    grade_point = 0.0
```

#### 5. 예쁜 출력 만들기

```python
print("="*40)
print("📊 학점 계산 결과")
print("="*40)
print(f"📝 입력 점수: {score}점")
print(f"🎓 학점: {grade}")
print(f"📈 GPA: {grade_point:.1f}")
print("="*40)
```

### 조건문 활용 팁

- **범위 조건**: `>=`, `<` 등을 활용하여 점수 구간 설정
- **논리 연산자**: `and`, `or`를 사용한 복합 조건
- **문자열 비교**: 사용자 입력 `y/n` 처리
- **중첩 조건문**: 장학금 판정 시 여러 조건 중첩 사용

### 사용자 친화적 프로그램 만들기

- 명확한 입력 안내 메시지
- 잘못된 입력에 대한 적절한 오류 메시지
- 시각적으로 보기 좋은 결과 출력
- 이모지를 활용한 재미있는 인터페이스

## 🎯 도전 과제

1. **상대평가 시스템**: 미리 정해진 기준으로 상대평가 구현
2. **학점 경고 시스템**: F학점에 대한 경고 메시지와 조치사항 안내
3. **누적 GPA 계산**: 여러 학기 성적을 고려한 전체 GPA 계산
4. **장학금 금액 계산**: 성적에 따른 장학금 금액 산정
