# 📊 학점 계산기

## 📋 프로젝트 개요

입력받은 점수를 바탕으로 학점을 계산하고, 장학금 대상 여부를 판단하는 실용적인 프로그램을 만들어보세요. 조건문을 활용하여 복잡한 판단 로직을 구현해봅시다!

## 🎯 학습 목표

- 다중 조건문(if-elif-else)을 활용한 복잡한 판단 로직 구현
- 논리 연산자를 이용한 복합 조건 처리
- 실생활 문제를 프로그래밍으로 해결하는 경험

## 🚀 기본 기능

### 핵심 요구사항

1. **점수 입력 및 검증**

   - 0~100점 범위의 점수 입력받기
   - 유효하지 않은 점수에 대한 처리

2. **학점 계산**

   - A: 90점 이상
   - B: 80점 이상 90점 미만
   - C: 70점 이상 80점 미만
   - D: 60점 이상 70점 미만
   - F: 60점 미만

3. **결과 출력**
   - 입력받은 점수와 해당하는 학점 표시
   - 깔끔한 출력 형태

## 🎨 추가 기능 1: 장학금 판정 시스템

### 요구사항

기본 학점 계산 위에 장학금 대상 여부를 추가로 판정합니다.

- **성적 우수 장학금**: A학점 (90점 이상)
- **성적 개선 장학금**: B학점 이상 (80점 이상)
- **격려 장학금**: C학점 이상 (70점 이상)

### 추가 기능

- 학점과 함께 해당하는 장학금 안내
- 장학금별 특징과 혜택 설명

## 💡 추가 기능 2: 상세 성적 분석

### 요구사항

기본 학점 계산과 장학금 판정을 확장하여 더 상세한 분석을 제공합니다.

- **성적 등급 세분화**: +, 0, - 구분 (예: A+, A0, A-)
- **다음 학점까지 필요 점수**: 더 높은 학점을 위해 필요한 점수
- **성적 상태 메시지**: 성적에 따른 격려 또는 조언

### 세분화된 학점 기준

- A+: 97점 이상, A0: 93점 이상, A-: 90점 이상
- B+: 87점 이상, B0: 83점 이상, B-: 80점 이상
- 이하 동일한 패턴으로 구성

## 🔥 추가 기능 3: 종합 성적 관리

### 요구사항

이전 모든 기능을 통합하여 종합적인 성적 관리 시스템을 구현합니다.

- **이전 학기 성적 입력**: 이전 학기 점수 입력받기
- **성적 변화 분석**: 이전 학기 대비 향상/하락 정도 계산
- **목표 설정**: 다음 학기 목표 학점 설정 및 필요 점수 계산
- **종합 조언**: 전체적인 학습 조언 제공

### 비교 분석

- 성적 향상도에 따른 추가 장학금 기회
- 목표 달성을 위한 구체적인 조언
- 학점 관리 팁 제공

## 💡 구현 힌트

### 필요한 파이썬 기능들

#### 1. 기본 조건문 구조

```python
# 기본 학점 계산 구조
score = int(input("점수를 입력하세요 (0-100): "))

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"
```

#### 2. 입력 검증 방법

```python
# 점수 범위 확인
if score < 0 or score > 100:
    print("잘못된 점수입니다.")
else:
    # 학점 계산 진행
```

#### 3. 장학금 판정 로직

```python
# 학점 계산 후 장학금 판정
if score >= 90:
    print("성적 우수 장학금 대상")
elif score >= 80:
    print("성적 개선 장학금 대상")
elif score >= 70:
    print("격려 장학금 대상")
```

### 조건문 활용 포인트

#### 1. 다중 조건문 (if-elif-else)

- **순서가 중요**: 높은 점수부터 낮은 점수 순으로 체크
- **범위 조건**: `>=`, `<` 등을 활용한 점수 구간 설정
- **명확한 기준**: 각 학점별 명확한 점수 기준 설정

#### 2. 논리 연산자 활용

- **and 연산자**: 여러 조건을 동시에 만족하는 경우
- **or 연산자**: 여러 조건 중 하나라도 만족하는 경우
- **not 연산자**: 조건을 반대로 확인하는 경우

#### 3. 중첩 조건문

- **복잡한 판정**: 장학금 판정 시 여러 조건을 중첩해서 체크
- **세부 분류**: 학점 내에서 +, 0, - 세분화
- **단계별 처리**: 먼저 기본 조건을 체크한 후 세부 조건 확인

### 세분화된 학점 계산 힌트

```python
# A학점 세분화 예시
if score >= 97:
    detailed_grade = "A+"
elif score >= 93:
    detailed_grade = "A0"
elif score >= 90:
    detailed_grade = "A-"
# 이하 B, C, D학점도 동일한 방식으로...
```

### 성적 비교 시스템 힌트

```python
# 이전 학기와 비교
current_score = int(input("현재 학기 점수: "))
previous_score = int(input("이전 학기 점수: "))

score_change = current_score - previous_score

if score_change > 0:
    print(f"{score_change}점 향상")
elif score_change < 0:
    print(f"{abs(score_change)}점 하락")
else:
    print("성적 동일")
```

- **단계별 진행**: 복잡한 기능도 단계별로 차근차근 진행

#### 2. 오류 처리

- **입력 검증**: 잘못된 입력에 대한 적절한 처리
- **범위 확인**: 점수 범위, 선택 범위 등 확인
- **친절한 오류 메시지**: 사용자가 이해하기 쉬운 오류 메시지

#### 3. 확장 가능한 구조

- **모듈화**: 기능별로 코드를 나누어 관리

- **점진적 개선**: 기본 기능부터 시작해서 점진적으로 기능 추가
