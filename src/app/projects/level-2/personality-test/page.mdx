# 🧠 미니 심리테스트

## 📋 프로젝트 개요

간단한 질문들을 통해 사용자의 성격 유형을 분석하고 재미있는 결과를 제공하는 심리테스트를 만들어보세요. 조건문을 활용하여 복잡한 성격 분석 로직을 구현해봅시다!

## 🎯 학습 목표

- 복잡한 조건문을 이용한 분기 처리
- 점수 기반 판정 시스템 구현
- 사용자와의 상호작용을 통한 재미있는 프로그램 제작

## 🚀 기본 기능

### 핵심 요구사항

1. **성격 유형 테스트**

   - 3가지 질문으로 성격 유형 판단
   - 각 질문에 대한 선택지 제공 (1, 2, 3번)
   - 선택에 따른 점수 누적

2. **성격 유형 결과**

   - 모험가형 (점수 7-9)
   - 안정추구형 (점수 4-6)
   - 신중형 (점수 3 이하)

3. **결과 출력**
   - 성격 유형과 특징 설명
   - 재미있는 조언이나 추천사항

## 🎨 추가 기능 1: 상세 성격 분석

### 요구사항

기본 성격 유형 판정에서 더 세밀한 분석을 추가합니다.

- **성격 유형 세분화**: 각 유형 내에서 상/중/하 등급 구분
- **강점과 약점 분석**: 각 성격 유형의 장단점 제시
- **추천 활동**: 성격 유형에 맞는 취미나 활동 추천

### 추가 분석 요소

- 사회성 지수 계산
- 도전 성향 분석
- 의사결정 스타일 파악

## 💡 추가 기능 2: 간단한 MBTI 테스트

### 요구사항

기본 성격 테스트를 확장하여 MBTI 스타일의 4차원 분석을 구현합니다.

- **4가지 질문**: 각 MBTI 차원별 한 문제씩
- **E vs I**: 외향성 vs 내향성
- **S vs N**: 감각형 vs 직관형
- **T vs F**: 사고형 vs 감정형
- **J vs P**: 판단형 vs 인식형

### 결과 제공

- 4글자 성격 유형 (예: ENFP, ISTJ)
- 각 차원별 설명
- 대표적인 특징 소개

## 🔥 추가 기능 3: 종합 성격 진단

### 요구사항

이전 모든 기능을 통합하여 종합적인 성격 진단을 제공합니다.

- **다중 영역 테스트**: 여러 관점에서 성격 분석
- **상세 점수 분석**: 각 영역별 점수와 비율 표시
- **성격 매칭**: 어울리는 성격 유형과 피해야 할 유형 제시
- **발전 방향 제시**: 개인 성장을 위한 구체적인 조언

### 종합 기능

- 성격 유형별 유명인 소개
- 추천 직업군 제시
- 연애 스타일 분석
- 스트레스 대처 방법 제안

## 💡 구현 힌트

### 필요한 파이썬 기능들

#### 1. 기본 질문과 점수 계산 구조

```python
# 질문과 선택지 제시
print("Q1. 주말에 가장 하고 싶은 활동은?")
print("1. 친구들과 파티나 모임 참가")
print("2. 새로운 장소 탐험이나 여행")
print("3. 집에서 영화보기나 독서")

choice1 = int(input("선택 (1, 2, 3): "))

# 점수 누적
score = choice1  # 첫 번째 질문 점수
# 이후 질문들도 동일한 방식으로 점수 추가
```

#### 2. 성격 유형 판정 로직

```python
# 총 점수로 성격 유형 결정
if score >= 7:
    personality_type = "모험가형"
elif score >= 4:
    personality_type = "안정추구형"
else:
    personality_type = "신중형"
```

#### 3. MBTI 스타일 분석 구조

```python
# 4차원 분석 예시
mbti_type = ""

# E vs I 판정
if ei_choice == 1:
    mbti_type += "E"
else:
    mbti_type += "I"

# S vs N 판정
if sn_choice == 1:
    mbti_type += "S"
else:
    mbti_type += "N"

# 최종 MBTI 유형: ENFP, ISTJ 등
```


#### 2. 복합 조건 처리

