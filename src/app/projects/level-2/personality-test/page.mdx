# 🧠 미니 심리테스트

## 📋 프로젝트 개요

간단한 질문들을 통해 사용자의 성격 유형을 분석하고 재미있는 결과를 제공하는 심리테스트를 만들어보세요. 조건문을 활용하여 복잡한 성격 분석 로직을 구현해봅시다!

## 🎯 학습 목표

- 복잡한 조건문을 이용한 분기 처리
- 점수 기반 판정 시스템 구현
- 사용자와의 상호작용을 통한 재미있는 프로그램 제작

## 🚀 기본 기능

### 핵심 요구사항

1. **성격 유형 테스트**

   - 3가지 질문으로 성격 유형 판단
   - 각 질문에 대한 선택지 제공 (1, 2, 3번)
   - 선택에 따른 점수 누적

2. **성격 유형 결과**

   - 모험가형 (점수 6-9)
   - 안정추구형 (점수 3-5)
   - 신중형 (점수 0-2)

3. **결과 출력**
   - 성격 유형과 특징 설명
   - 재미있는 조언이나 추천사항

## 🎨 추가 기능 1: MBTI 간이 테스트

### 요구사항

각 MBTI 차원별로 한 문제씩 출제하여 4글자 성격 유형 도출

- **E(외향) vs I(내향)**: 에너지 충전 방식
- **S(감각) vs N(직관)**: 정보 수집 방식
- **T(사고) vs F(감정)**: 의사결정 방식
- **J(판단) vs P(인식)**: 생활 태도

## 💡 추가 기능 2: 상세 분석 및 점수

### 요구사항

- 각 영역별 상세 점수 표시
- 강점과 개선점 분석
- 추천 직업이나 취미 제안
- 같은 유형의 유명인 소개

## 💡 구현 힌트

### 필요한 파이썬 기능들

#### 1. 질문과 선택지 제시

```python
print("🧠 성격 유형 테스트에 오신 것을 환영합니다!")
print("="*50)

print("Q1. 주말에 가장 하고 싶은 활동은?")
print("1. 친구들과 파티나 모임 참가")
print("2. 새로운 장소 탐험이나 여행")
print("3. 집에서 영화보기나 독서")

choice1 = int(input("선택 (1, 2, 3): "))
```

#### 2. 선택에 따른 점수 계산

```python
score = 0

# 첫 번째 질문 점수 계산
if choice1 == 1:
    score += 3
elif choice1 == 2:
    score += 2
else:
    score += 1

# 추가 질문들도 동일한 방식으로...
```

#### 3. 점수 기반 성격 유형 판단

```python
if score >= 6:
    personality_type = "모험가형"
    description = "새로운 경험을 좋아하고 활동적인 성격입니다."
    advice = "다양한 취미활동에 도전해보세요!"
elif score >= 3:
    personality_type = "안정추구형"
    description = "균형잡힌 사고와 안정을 추구하는 성격입니다."
    advice = "꾸준한 계획을 세워 목표를 달성해보세요!"
else:
    personality_type = "신중형"
    description = "깊이 있게 생각하고 신중한 판단을 하는 성격입니다."
    advice = "충분한 계획을 세운 후 행동하는 것이 좋겠어요!"
```

#### 4. MBTI 판정

```python
mbti = ""

# E vs I 판정
print("Q1. 에너지를 얻는 방법은?")
print("1. 사람들과 어울리기")
print("2. 혼자만의 시간 갖기")
ei_choice = int(input("선택: "))

if ei_choice == 1:
    mbti += "E"
else:
    mbti += "I"

# S vs N, T vs F, J vs P도 동일한 방식으로...
```

#### 5. MBTI 유형별 설명

```python
mbti_descriptions = {
    "ENFP": "열정적인 활동가 - 창의적이고 사교적",
    "INFP": "열정적인 중재자 - 이상주의적이고 충성심 강함",
    "ENTJ": "대담한 통솔자 - 타고난 리더",
    "INTJ": "용의주도한 전략가 - 독립적이고 계획적",
    # ... 16가지 유형 모두
}

if mbti in mbti_descriptions:
    print(f"당신의 MBTI: {mbti}")
    print(f"특징: {mbti_descriptions[mbti]}")
```

#### 6. 결과 출력 꾸미기

```python
print("\n" + "🎭" * 20)
print("     심리테스트 결과")
print("🎭" * 20)
print(f"✨ 성격 유형: {personality_type}")
print(f"📝 설명: {description}")
print(f"💡 조언: {advice}")
print("🎭" * 20)
```
