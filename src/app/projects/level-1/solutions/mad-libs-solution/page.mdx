# 🎭 Mad Libs (답안)

## 🚀 기본 기능 답안

```python
# 🎭 Mad Libs 게임
print("🎭 Mad Libs 게임에 오신 것을 환영합니다!")
print("재미있는 이야기를 만들어보세요!")
print("="*50)

# 단어 수집
name = input("이름: ")
adjective1 = input("형용사 1: ")
noun1 = input("명사 1: ")
verb1 = input("동사 1: ")
adjective2 = input("형용사 2: ")
noun2 = input("명사 2: ")
number = input("숫자: ")

# 이야기 생성
print("\n" + "="*50)
print("🎪 완성된 이야기")
print("="*50)

story = f"""
옛날 옛적에 {adjective1}한 {name}이(가) 살고 있었습니다.
{name}은(는) 매일 {noun1}을(를) {verb1}하는 것을 좋아했습니다.
어느 날, {adjective2}한 {noun2}이(가) 나타나서 말했습니다:
"{name}님, {number}개의 소원을 들어드리겠습니다!"
그리하여 {name}은(는) 행복하게 살았답니다.
"""

print(story)
print("="*50)
print("🎉 이야기가 완성되었습니다!")
```

## 🌟 추가 기능 1: 다양한 이야기 템플릿

```python
# 🎭 Mad Libs 게임 (다양한 템플릿)
print("🎭 Mad Libs 게임 - 다양한 템플릿")
print("="*50)

# 기본 이야기
print("📖 기본 이야기")
print("-"*20)
name = input("이름: ")
adjective1 = input("형용사 1: ")
noun1 = input("명사 1: ")
verb1 = input("동사 1: ")

basic_story = f"""
🌟 {name}의 이야기
{adjective1}한 {name}이(가) {noun1}을(를) {verb1}했습니다.
"""
print(basic_story)

# 모험 이야기
print("🏰 모험 이야기")
print("-"*20)
hero_name = input("주인공 이름: ")
place = input("장소: ")
monster = input("괴물 이름: ")
weapon = input("무기: ")
treasure = input("보물: ")
adjective_scary = input("무서운 형용사: ")
sound = input("의성어: ")

adventure_story = f"""
🏰 {hero_name}의 대모험

용감한 {hero_name}은(는) {place}에 도착했습니다.
갑자기 {adjective_scary}한 {monster}이(가) 나타나서 "{sound}!" 소리를 냈습니다.
{hero_name}은(는) {weapon}을(를) 들고 용감하게 싸웠습니다.
결국 {monster}을(를) 물리치고 {treasure}을(를) 발견했습니다!
"""

print(adventure_story)
```

## 🎨 추가 기능 2: 인터랙티브 스토리

```python
# 🎭 Mad Libs 게임 (인터랙티브 스토리)
print("🎭 Mad Libs 게임 - 인터랙티브 스토리")
print("="*50)

# 기본 템플릿
print("📖 기본 이야기 만들기")
name = input("이름: ")
adjective = input("형용사: ")
noun = input("명사: ")

print(f"\n✨ {adjective}한 {name}이(가) {noun}을(를) 발견했습니다!")

# 인터랙티브 스토리
print("\n🎨 나만의 캐릭터 만들기")
print("-"*30)

# 캐릭터 설정
character_name = input("캐릭터 이름: ")
job = input("직업: ")
hobby = input("취미: ")
favorite_food = input("좋아하는 음식: ")
dream = input("꿈: ")

# 상황 설정
problem = input("해결해야 할 문제: ")
solution = input("해결 방법: ")
result = input("결과: ")

# 인터랙티브 스토리 생성
interactive_story = f"""
🎭 {character_name}의 하루

{job}인 {character_name}은(는) {hobby}을(를) 즐기며 살고 있었습니다.
{favorite_food}을(를) 먹으며 {dream}을(를) 꿈꾸고 있었죠.

그런데 어느 날, {problem}라는 문제가 생겼습니다!
{character_name}은(는) 고민 끝에 {solution}을(를) 시도했습니다.
그 결과 {result}하게 되었답니다.

{character_name}은(는) 이 경험을 통해 더욱 성장했습니다!
"""

print(interactive_story)
```

## 🎪 추가 기능 3: 재미있는 시나리오

```python
# 🎭 Mad Libs 게임 (완전체 버전)
print("🎭 Mad Libs 게임 - 완전체 버전")
print("="*50)

# 기본 이야기
print("📖 짧은 이야기")
name = input("이름: ")
adjective = input("형용사: ")
print(f"✨ {adjective}한 {name}의 모험이 시작됩니다!")

# 학교 생활 시나리오
print("\n🎪 학교 생활 이야기")
print("-"*30)

# 학교 관련 단어 수집
student_name = input("학생 이름: ")
subject = input("과목: ")
teacher_name = input("선생님 이름: ")
grade = input("점수: ")
excuse = input("변명: ")
friend_name = input("친구 이름: ")
snack = input("간식: ")

# 학교 이야기 생성
school_story = f"""
🏫 {student_name}의 학교 생활

{student_name}은(는) {subject} 시간에 {teacher_name} 선생님께
{grade}점을 받았습니다.
"{excuse}라서 그렇습니다!" 라고 말했더니
선생님이 웃으셨습니다.

쉬는 시간에 {friend_name}과(와) 함께 {snack}을(를) 먹으며
"다음에는 더 열심히 하자!"고 다짐했답니다.
"""

print(school_story)

# 가족 이야기
print("\n👨‍👩‍👧‍👦 가족 이야기")
print("-"*20)

family_member = input("가족 구성원: ")
activity = input("활동: ")
place = input("장소: ")
food = input("음식: ")
feeling = input("감정: ")

family_story = f"""
👨‍👩‍👧‍👦 우리 가족 이야기

주말에 {family_member}과(와) 함께 {place}에서 {activity}을(를) 했습니다.
{food}을(를) 먹으며 {feeling}한 시간을 보냈어요.
우리 가족은 정말 행복합니다!
"""

print(family_story)
```

## 💡 핵심 포인트

1. **문자열 조작**: f-string을 활용한 변수 삽입
2. **여러 줄 문자열**: `"""` 사용으로 긴 이야기 작성
3. **사용자 입력**: `input()` 함수로 다양한 단어 수집
4. **창의적 구성**: 시작-중간-끝 구조로 완성도 있는 이야기 작성

## 🎯 추가 아이디어

- **계절별 이야기**: 봄, 여름, 가을, 겨울 테마
- **직업별 이야기**: 의사, 선생님, 요리사 등
- **동물 이야기**: 애완동물이나 야생동물 테마
- **우주 이야기**: 외계인이나 우주 탐험 테마
