# 💼 나만의 명함 만들기

## 📋 프로젝트 개요

변수와 입출력을 활용하여 나만의 개성있는 디지털 명함을 만들어보세요. ASCII 아트와 함께 예쁘게 꾸며보세요!

## 🎯 학습 목표

- 변수를 활용한 정보 저장 및 관리
- 문자열 조작과 포맷팅 기법
- 사용자 입력 처리
- ASCII 아트와 출력 디자인

## 🚀 기본 기능

### 핵심 요구사항

1. **개인 정보 입력**

   - 이름, 나이, 직업, 취미 입력받기
   - 연락처 정보 (이메일, 전화번호) 입력받기

2. **명함 출력**
   - 입력받은 정보를 예쁘게 정리하여 출력
   - 기본적인 테두리나 구분선 사용

### 💻 기본 구현 예시

```python
# 💼 나만의 명함 만들기
print("💼 나만의 디지털 명함 만들기")
print("="*40)

# 개인 정보 입력
name = input("이름: ")
age = int(input("나이: "))
job = input("직업: ")
hobby = input("취미: ")
email = input("이메일: ")
phone = input("전화번호: ")

# 명함 출력
print("\n" + "="*40)
print("✨ 나만의 디지털 명함 ✨")
print("="*40)
print(f"👤 이름: {name}")
print(f"🎂 나이: {age}세")
print(f"💼 직업: {job}")
print(f"🎨 취미: {hobby}")
print(f"📧 이메일: {email}")
print(f"📱 전화번호: {phone}")
print("="*40)
print("감사합니다! 😊")
```

## 🎨 추가 기능 1: ASCII 아트 장식

### 요구사항

- 명함에 ASCII 아트 장식 추가
- 다양한 기호와 문자로 예쁘게 꾸미기

### 구현 예시

```python
# ASCII 아트 장식
print("┌─────────────────────────────────────┐")
print("│        ✨ 디지털 명함 ✨           │")
print("├─────────────────────────────────────┤")
print(f"│ 👤 이름: {name:<25} │")
print(f"│ 🎂 나이: {age}세{'':<22} │")
print(f"│ 💼 직업: {job:<25} │")
print(f"│ 🎨 취미: {hobby:<25} │")
print("├─────────────────────────────────────┤")
print(f"│ 📧 {email:<30} │")
print(f"│ 📱 {phone:<30} │")
print("└─────────────────────────────────────┘")
```

## 🌟 추가 기능 2: 개성있는 명함 스타일

### 요구사항

- 명함 스타일 선택 기능
- 좋아하는 색상이나 테마 반영
- 개인 슬로건이나 좌우명 추가

### 구현 예시

```python
# 추가 정보 입력
favorite_color = input("좋아하는 색상: ")
motto = input("좌우명 또는 슬로건: ")

# 개성있는 명함 출력
print("\n" + "🎨"*20)
print(f"🌈 {favorite_color.upper()} 테마 명함 🌈")
print("🎨"*20)
print(f"✨ {name}님의 명함 ✨")
print(f"💫 \"{motto}\"")
print("-"*40)
# ... 나머지 정보 출력
```

## 💡 구현 힌트

### 문자열 포맷팅 팁

- `f-string` 사용하여 변수 삽입
- 문자열 정렬: `{변수:<길이}` (왼쪽 정렬)
- 반복 문자: `"="*개수`

### 예쁜 출력 팁

- 적절한 공백과 줄바꿈 활용
- 이모지 사용으로 시각적 효과 증대
- 일관된 스타일과 정렬 유지

## 🎯 도전 과제

1. **다국어 명함**: 한국어와 영어 명함 동시 출력
2. **명함 템플릿**: 여러 가지 명함 스타일 중 선택
3. **명함 저장**: 파일로 저장하는 기능 (추후 학습)
4. **명함 비교**: 친구 명함과 비교하는 기능
