# 🎭 Mad Libs 게임

## 📋 프로젝트 개요

명사, 동사, 형용사 등을 입력받아 재미있는 이야기를 완성하는 Mad Libs 게임을 만들어보세요. 창의적인 단어 조합으로 웃긴 이야기를 만들어보세요!

## 🎯 학습 목표

- 변수를 활용한 문자열 조작
- 사용자 입력 처리
- 문자열 포맷팅과 템플릿 활용
- 창의적 프로그래밍 사고

## 🚀 기본 기능

### 핵심 요구사항

1. **단어 수집**

   - 명사, 동사, 형용사 등 입력받기
   - 이야기 템플릿에 필요한 단어들 수집

2. **이야기 생성**
   - 입력받은 단어들을 템플릿에 삽입
   - 완성된 이야기 출력

### 💻 기본 구현 예시

```python
# 🎭 Mad Libs 게임
print("🎭 Mad Libs 게임에 오신 것을 환영합니다!")
print("재미있는 이야기를 만들어보세요!")
print("="*50)

# 단어 수집
name = input("이름: ")
adjective1 = input("형용사 1: ")
noun1 = input("명사 1: ")
verb1 = input("동사 1: ")
adjective2 = input("형용사 2: ")
noun2 = input("명사 2: ")
number = input("숫자: ")

# 이야기 생성
print("\n" + "="*50)
print("🎪 완성된 이야기")
print("="*50)

story = f"""
옛날 옛적에 {adjective1}한 {name}이(가) 살고 있었습니다.
{name}은(는) 매일 {noun1}을(를) {verb1}하는 것을 좋아했습니다.
어느 날, {adjective2}한 {noun2}이(가) 나타나서 말했습니다:
"{name}님, {number}개의 소원을 들어드리겠습니다!"
그리하여 {name}은(는) 행복하게 살았답니다.
"""

print(story)
print("="*50)
print("🎉 이야기가 완성되었습니다!")
```

## 🌟 추가 기능 1: 다양한 이야기 템플릿

### 요구사항

- 여러 개의 이야기 템플릿 제공
- 모험, 로맨스, 판타지 등 다양한 장르
- 더 많은 단어 타입 활용

### 구현 예시

```python
# 🌟 다양한 이야기 템플릿
print("\n🌟 모험 이야기")
print("="*30)

# 모험 이야기용 단어 수집
hero_name = input("주인공 이름: ")
place = input("장소: ")
monster = input("괴물 이름: ")
weapon = input("무기: ")
treasure = input("보물: ")
adjective_scary = input("무서운 형용사: ")
sound = input("의성어: ")

# 모험 이야기 생성
adventure_story = f"""
🏰 {hero_name}의 대모험

용감한 {hero_name}은(는) {place}에 도착했습니다.
갑자기 {adjective_scary}한 {monster}이(가) 나타나서 "{sound}!" 소리를 냈습니다.
{hero_name}은(는) {weapon}을(를) 들고 용감하게 싸웠습니다.
결국 {monster}을(를) 물리치고 {treasure}을(를) 발견했습니다!
"""

print(adventure_story)
```

## 🎨 추가 기능 2: 인터랙티브 스토리

### 요구사항

- 사용자의 선택에 따라 이야기 변화
- 캐릭터 설정 기능
- 이야기 중간에 추가 선택지

### 구현 예시

```python
# 🎨 인터랙티브 스토리
print("\n🎨 나만의 캐릭터 만들기")
print("="*30)

# 캐릭터 설정
character_name = input("캐릭터 이름: ")
job = input("직업: ")
hobby = input("취미: ")
favorite_food = input("좋아하는 음식: ")
dream = input("꿈: ")

# 상황 설정
problem = input("해결해야 할 문제: ")
solution = input("해결 방법: ")
result = input("결과: ")

# 인터랙티브 스토리 생성
interactive_story = f"""
🎭 {character_name}의 하루

{job}인 {character_name}은(는) {hobby}을(를) 즐기며 살고 있었습니다.
{favorite_food}을(를) 먹으며 {dream}을(를) 꿈꾸고 있었죠.

그런데 어느 날, {problem}라는 문제가 생겼습니다!
{character_name}은(는) 고민 끝에 {solution}을(를) 시도했습니다.
그 결과 {result}하게 되었답니다.

{character_name}은(는) 이 경험을 통해 더욱 성장했습니다!
"""

print(interactive_story)
```

## 🎪 추가 기능 3: 재미있는 시나리오

### 요구사항

- 일상생활 기반 웃긴 시나리오
- 학교, 가족, 친구 관련 이야기
- 더 많은 변수와 복잡한 스토리

### 구현 예시

```python
# 🎪 학교 생활 시나리오
print("\n🎪 학교 생활 이야기")
print("="*30)

# 학교 관련 단어 수집
student_name = input("학생 이름: ")
subject = input("과목: ")
teacher_name = input("선생님 이름: ")
grade = input("점수: ")
excuse = input("변명: ")
friend_name = input("친구 이름: ")
snack = input("간식: ")

# 학교 이야기 생성
school_story = f"""
🏫 {student_name}의 학교 생활

{student_name}은(는) {subject} 시간에 {teacher_name} 선생님께
{grade}점을 받았습니다.
"{excuse}라서 그렇습니다!" 라고 말했더니
선생님이 웃으셨습니다.

쉬는 시간에 {friend_name}과(와) 함께 {snack}을(를) 먹으며
"다음에는 더 열심히 하자!"고 다짐했답니다.
"""

print(school_story)
```

## 💡 구현 힌트

### 문자열 처리 팁

- `f-string` 사용으로 가독성 향상
- 여러 줄 문자열은 `"""` 사용
- 적절한 조사 처리 (은/는, 이/가, 을/를)

### 이야기 작성 팁

- 시작-중간-끝 구조 잡기
- 감정 표현 단어 활용
- 적절한 길이로 조절

## 🎯 도전 과제

1. **랜덤 이야기**: 여러 템플릿 중 랜덤 선택
2. **이야기 저장**: 완성된 이야기 파일로 저장
3. **단어 검증**: 입력된 단어가 적절한지 확인
4. **이야기 체인**: 여러 이야기를 연결하는 기능
