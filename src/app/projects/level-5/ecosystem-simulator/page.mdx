# 🌱 가상 생태계 시뮬레이터

식물과 동물이 공생하는 생태계를 만들고 관리하는 시뮬레이션 게임을 만들어보세요!

## 📋 프로젝트 설명

이 프로젝트에서는 가상의 생태계를 만들고 관리하는 프로그램을 개발합니다. 식물을 키우고, 동물을 추가하고, 생태계의 균형을 맞춰보세요.

## 🚀 1단계: 식물 관리 시스템

기본적인 식물 키우기 시스템을 만들어보세요.

### 📝 구현할 기능
- 새로운 식물 심기
- 물주기 (물 레벨 증가)
- 영양분 주기 (성장 속도 증가)
- 식물 상태 확인
- 시간 경과 시 자동 성장/물 감소

### 💻 예상 출력
```
🌱 가상 생태계 시뮬레이터에 오신 것을 환영합니다!
========================================

=== 나의 생태계 ===
1. 식물 심기
2. 물주기
3. 영양분 주기
4. 식물 상태 보기
5. 시간 경과
6. 종료

선택: 1

=== 식물 심기 ===
식물 이름: 장미
🌱 장미가 심어졌습니다!

=== 나의 생태계 ===
...
선택: 4

=== 식물 상태 ===
🌱 장미 - 건강도: 100% | 크기: 20cm | 물: 80%
   성장 단계: 새싹 🌱
```

## 🔥 2단계: 동물 친구들

생태계에 동물들을 추가해보세요.

### 📝 구현할 기능
- 동물 추가 (토끼, 나비, 새 등)
- 동물 먹이주기
- 동물과 식물의 상호작용
- 생태계 균형 관리

### 💻 예상 출력
```
=== 나의 생태계 ===
1. 식물 심기     2. 동물 추가     3. 물주기
4. 먹이주기      5. 생태계 보기    6. 시간 경과
7. 종료

선택: 2

=== 동물 추가 ===
1. 토끼 🐰 (채식)
2. 나비 🦋 (꽃가루)
3. 새 🐦 (씨앗)
동물 선택: 1

토끼 이름: 뽀로로
🐰 뽀로로가 생태계에 추가되었습니다!

선택: 5

=== 나의 생태계 ===
🌸 장미 (성인) - 건강: 85% | 크기: 80cm | 물: 60%
🐰 뽀로로 - 행복도: 90% | 배고픔: 30% | 에너지: 70%

생태계 균형도: 85% 🌿
```

## 🎖️ 3단계: 생태계 균형

생태계의 균형을 맞추고 다양한 이벤트를 처리해보세요.

### 📝 구현할 기능
- 날씨 시스템 (맑음, 비, 흐림)
- 계절 변화 효과
- 생태계 균형 점수
- 특별 이벤트 (꽃 피기, 새끼 동물 등)
- 생태계 통계

### 💻 예상 출력
```
=== 나의 생태계 ===
날씨: ☀️ 맑음 | 계절: 🌸 봄 | 균형도: 92%

1. 식물 관리     2. 동물 관리     3. 생태계 보기
4. 통계 보기     5. 시간 경과     6. 종료

선택: 3

=== 🌿 생태계 현황 ===
🌸 장미 (성인) - 건강: 95% | 꽃: 3개 🌺
🌿 민들레 (청년) - 건강: 80% | 씨앗: 5개
🐰 뽀로로 - 행복: 95% | 만족도: 높음
🦋 나비 - 활동도: 85% | 꽃가루 수집: 활발

🎉 특별 이벤트: 장미에서 예쁜 꽃이 피었습니다!

생태계 균형도: 92% 
- 식물 다양성: 우수 ⭐⭐⭐
- 동물 만족도: 높음 ⭐⭐⭐
- 자원 순환: 원활 ⭐⭐⭐

선택: 4

=== 📊 생태계 통계 ===
운영 일수: 15일
총 식물: 3종 (장미, 민들레, 클로버)
총 동물: 2마리 (토끼 1, 나비 1)
생태계 점수: 920점

🏆 달성한 업적:
✅ 첫 번째 꽃 피우기
✅ 동물 친구 만들기
✅ 생태계 균형 90% 달성
```

### 중요 개념

- **리스트**: 여러 생물체 관리
- **조건문**: 날씨와 계절에 따른 변화
- **반복문**: 시간 경과에 따른 자동 처리
