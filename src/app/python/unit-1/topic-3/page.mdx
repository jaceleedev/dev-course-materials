# Topic 3: 파이썬 설치하기 🛠️

## 🎯 학습 목표

파이썬을 운영체제에 맞게 올바르게 설치하고, 정상 작동을 확인하며, 발생할 수 있는 문제들을 해결할 수 있습니다.

## 🖥️ 시작하기 전에: 내 컴퓨터 정보 확인

### Windows 사용자

1. `윈도우 키 + R` → `winver` 입력 → 엔터
   - **winver**는 "Windows Version"의 줄임말로, 현재 설치된 Windows 버전 정보를 보여주는 명령어입니다
2. Windows 10 이상 권장 (Windows 8.1도 가능하지만 보안 업데이트 종료)

### Mac 사용자

1. `Apple 메뉴` → `이 Mac에 관하여`
2. macOS 10.15 (Catalina) 이상 권장

### Linux 사용자

```bash
lsb_release -a  # Ubuntu/Debian
cat /etc/os-release  # 대부분의 배포판
```

## 📥 파이썬 버전 이해하기

### 현재 지원되는 파이썬 버전 (2025년 6월 기준)

- **Python 3.13.5**: 최신 안정 버전 (2025년 6월 출시, 추천 ⭐)
- **Python 3.12.11**: 널리 사용되는 안정 버전
- **Python 3.11.13**: 여전히 지원되는 버전

### 🚨 중요: Python 2 vs Python 3의 차이

#### Python 2 (더 이상 사용하지 마세요!)

- **2020년 1월 1일 공식 지원 종료**
- 보안 업데이트 없음
- 새로운 라이브러리들이 지원하지 않음
- 레거시 코드에서만 제한적으로 사용

#### Python 3 (현재 표준!)

- **2008년 출시 이후 현재까지 활발히 개발**
- 모든 새로운 기능과 라이브러리가 Python 3 기준
- 더 나은 Unicode 지원
- 향상된 성능과 보안

> **반드시 Python 3를 설치하세요!** Python 2는 역사적 가치만 있을 뿐, 현재는 사용하지 않습니다.

## 🪟 Windows 설치 가이드

### 공식 웹사이트에서 설치 (권장)

#### 1단계: 다운로드

1. [python.org](https://www.python.org/downloads/) 접속
2. "Download Python 3.13.5" 노란색 버튼 클릭
3. 파일이 다운로드될 때까지 대기

#### 2단계: 설치 실행

1. 다운로드한 파일 실행 (`python-3.13.5-amd64.exe`)
2. **🚨 중요**: 다음 두 체크박스를 설정하세요!

```
┌─────────────────────────────────────┐
│ Python 3.13.5 Setup                 │
├─────────────────────────────────────┤
│ ☑ Use admin privileges when         │  ← 체크 권장
│   installing py.exe                 │
│ ☑ Add python.exe to PATH            │  ← 반드시 체크!
│                                     │
│ [Install Now]  [Customize]          │
└─────────────────────────────────────┘
```

#### 🔍 체크박스 옵션 설명

**Add python.exe to PATH (반드시 체크!):**

- **PATH**란? 운영체제가 실행 파일을 찾는 폴더 목록입니다
- 체크하면: 어느 폴더에서든 `python` 명령어로 파이썬을 실행할 수 있습니다
- 체크하지 않으면: 파이썬 설치 폴더로 이동해야만 실행 가능 (매우 불편!)

**Use admin privileges when installing py.exe (체크 권장):**

- **py.exe**는 파이썬 런처로, 여러 파이썬 버전을 관리하는 도구입니다
- 체크하면: 모든 사용자가 `py` 명령어를 사용할 수 있습니다
- 체크하지 않아도: 개인 사용자로만 설치되어 문제없이 작동합니다
- **권장**: 체크 (나중에 다른 파이썬 버전 설치 시 편리함)

3. "Install Now" 클릭
4. 관리자 권한 허용
5. 설치 완료까지 대기 (약 2-5분)

#### 3단계: 설치 확인

1. `윈도우 키 + R` → `cmd` 입력 → 엔터
2. 명령 프롬프트에서 다음 입력:

```bash
python --version
```

성공 시 출력:

```
Python 3.13.5
```

## 🍎 macOS 설치 가이드

### 사전 확인: 기본 설치된 Python 확인

macOS에는 Python이 기본 설치되어 있지만, 보통 구버전입니다:

```bash
python3 --version
```

Python 3.10 이하가 나오면 최신 버전 설치가 필요합니다.

### 방법 1: 공식 웹사이트

#### 1단계: 다운로드

1. [python.org](https://www.python.org/downloads/) 접속
2. 상단 메뉴에서 **"Downloads"** 클릭
3. **"macOS"** 클릭
4. "Latest Python 3 Release - Python 3.13.5" 클릭
5. "macOS 64-bit universal2 installer" 다운로드

#### 2단계: 설치

1. 다운로드한 `.pkg` 파일 실행
2. 설치 마법사 따라가기
3. 관리자 비밀번호 입력

#### 3단계: 터미널에서 확인

```bash
python3 --version
pip3 --version
```

### 방법 2: Homebrew (개발자 추천)

Homebrew는 macOS용 패키지 관리자입니다.

#### 1단계: Homebrew 설치

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

#### 2단계: Python 설치

```bash
brew install python
```

#### 3단계: 확인

```bash
python3 --version
brew list python
```

**Homebrew 장점:**

- 최신 버전 자동 업데이트
- 의존성 관리 자동
- 개발 도구들과 잘 연동

## 🐧 Linux 설치 가이드

대부분의 Linux 배포판에는 Python 3가 기본 설치되어 있습니다.

### Ubuntu/Debian 계열

#### 현재 버전 확인

```bash
python3 --version
```

#### 최신 버전 설치

```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv
```

### CentOS/RHEL/Fedora 계열

#### CentOS/RHEL:

```bash
sudo yum install python3 python3-pip
```

#### Fedora:

```bash
sudo dnf install python3 python3-pip
```

### Arch Linux 계열

```bash
sudo pacman -S python python-pip
```

## ✅ 설치 확인 및 테스트

### 기본 확인 명령어

모든 운영체제에서 다음 명령어들이 정상 작동해야 합니다:


## 🚨 자주 발생하는 문제와 해결법

### 문제 1: "python은 내부 또는 외부 명령이 아닙니다" (Windows)

**원인**: PATH 환경변수에 Python이 등록되지 않음

**해결법**: 파이썬 재설치 시 "Add Python to PATH" 옵션 체크

### 문제 2: Mac에서 "python" vs "python3" 혼동

**현상**: `python` 명령어가 Python 2를 실행

**해결법**: 별칭(alias) 설정

```bash
# ~/.zshrc 또는 ~/.bash_profile에 추가
echo "alias python=python3" >> ~/.zshrc
echo "alias pip=pip3" >> ~/.zshrc
source ~/.zshrc
```

### 문제 3: 권한 오류 (Linux/Mac)

**현상**: "Permission denied" 오류

**해결법**:

```bash
# pip 사용 시
pip3 install --user 패키지명

# 또는 가상환경 사용 (권장)
python3 -m venv myenv
source myenv/bin/activate  # Linux/Mac
# myenv\Scripts\activate  # Windows
```
