### 🤖 기억 없는 프로그램의 한계

중요한 데이터를 기억하지 못한다면..?

```python

```

**문제점**: 중요한 정보를 기억하지 못해서 프로그램이 문제를 일으킬 수 있어요!

### 🌟 변수: 프로그램의 기억력

**변수(Variable)**는 컴퓨터의 메모리에 만드는 **마법의 상자**입니다:

- 소중한 정보를 **안전하게 보관**
- 필요할 때마다 **꺼내서 사용**
- 내용을 **바꾸거나 업데이트** 가능

> **변수의 철학**:
> "기억한다, 고로 존재한다"
>
> 변수가 있어야 프로그램이 **과거를 기억하고, 현재를 인식하고, 미래를 계획**할 수 있어요! 🧠✨

## 📦 PART 1: 변수의 기본 - 상자 만들고 사용하기

### 첫 번째 변수 만들기

```python
# 🎁 선물상자 만들기

is_student = True

print(f"이름: {name}")

print(f"키: {height}cm")
print(f"학생인가요? {is_student}")
```

### 🔍 변수 해부하기

```python
name = "김파이썬"
#  ↑     ↑      ↑
# 상자   내용물   상자에 넣기
```

1. **`name`**: 상자의 이름표 (변수명)
2. **`=`**: 상자에 넣는 동작 (할당 연산자)
3. **`"김파이썬"`**: 상자에 들어갈 내용 (값)

## 🔄 PART 3: 변수값 변경 - 상자 내용물 바꾸기

### 변수는 변할 수 있어요!

```python
# 📈 성장하는 게임 캐릭터

print(f"🎮 {player_name} (Lv.{player_level})")

print(f"💰 Gold: {player_gold}")

print("\n⬆️ 레벨업!")
player_level = player_level + 1
player_hp = 100  # 완전 회복

print(f"🎮 {player_name} (Lv.{player_level})")
print(f"❤️ HP: {player_hp}")
print(f"💰 Gold: {player_gold}")
```

## 🎯 PART 4: 실전 프로젝트 - 변수로 만드는 지능형 프로그램

### 프로젝트 1: 개인 성장 추적기

```python
# 📊 개인 성장 추적기
print("🌱 개인 성장 추적기")
print("="*40)

# 기본 정보 수집
name = input("이름: ")
current_age = int(input("현재 나이: "))

# 현재 상태
current_weight = float(input("현재 몸무게(kg): "))
current_height = float(input("현재 키(cm): "))
study_hours_today = int(input("오늘 공부한 시간: "))

# 목표 설정
target_weight = float(input("목표 몸무게(kg): "))
daily_study_goal = int(input("일일 공부 목표(시간): "))

# BMI 계산
height_m = current_height / 100
current_bmi = current_weight / (height_m * height_m)

# 목표까지의 차이 계산
weight_difference = current_weight - target_weight
study_achievement = (study_hours_today / daily_study_goal) * 100

# 보고서 출력
print("\n" + "📊 성장 보고서")
print("="*40)
print(f"👤 {name}님 ({current_age}세)")
print(f"📏 키: {current_height}cm")
print(f"⚖️ 현재 몸무게: {current_weight}kg")
print(f"🎯 목표 몸무게: {target_weight}kg")
print(f"📈 BMI: {current_bmi:.1f}")

if weight_difference > 0:
    print(f"💪 목표까지 {weight_difference:.1f}kg 감량 필요")
elif weight_difference < 0:
    print(f"🍎 목표까지 {-weight_difference:.1f}kg 증량 필요")
else:
    print("🎉 목표 달성!")

print(f"📚 오늘 공부: {study_hours_today}시간")
print(f"🎯 달성률: {study_achievement:.1f}%")

if study_achievement >= 100:
    print("🏆 오늘 목표 달성! 축하합니다!")
else:
    remaining_hours = daily_study_goal - study_hours_today
    print(f"⏰ {remaining_hours}시간 더 공부하면 목표 달성!")
```

### 프로젝트 2: 간단한 은행 계좌 시뮬레이터

```python
# 🏦 간단한 은행 계좌 시뮬레이터
print("🏦 파이썬 은행에 오신 것을 환영합니다!")
print("="*50)

# 계좌 개설
account_name = input("계좌 이름: ")
initial_balance = int(input("초기 입금액: "))

# 계좌 정보
balance = initial_balance
transaction_count = 0

print(f"\n🎉 {account_name}님의 계좌가 개설되었습니다!")
print(f"💰 잔액: {balance:,}원")

while True:
    print("\n" + "="*30)
    print("🏦 파이썬 은행 메뉴")
    print("="*30)
    print("1. 잔액 조회")
    print("2. 입금")
    print("3. 출금")
    print("4. 거래 내역")
    print("5. 종료")

    choice = input("선택 (1-5): ")

    if choice == "1":
        print(f"\n💰 현재 잔액: {balance:,}원")

    elif choice == "2":
        deposit = int(input("입금할 금액: "))
        balance += deposit
        transaction_count += 1
        print(f"✅ {deposit:,}원이 입금되었습니다.")
        print(f"💰 현재 잔액: {balance:,}원")

    elif choice == "3":
        withdraw = int(input("출금할 금액: "))
        if withdraw <= balance:
            balance -= withdraw
            transaction_count += 1
            print(f"✅ {withdraw:,}원이 출금되었습니다.")
            print(f"💰 현재 잔액: {balance:,}원")
        else:
            print("❌ 잔액이 부족합니다!")

    elif choice == "4":
        print(f"\n📊 거래 내역")
        print(f"계좌명: {account_name}")
        print(f"초기 금액: {initial_balance:,}원")
        print(f"현재 잔액: {balance:,}원")
        print(f"총 거래 횟수: {transaction_count}회")

    elif choice == "5":
        print(f"\n👋 {account_name}님, 이용해주셔서 감사합니다!")
        break

    else:
        print("❌ 잘못된 선택입니다.")
```

## 🚨 자주 발생하는 변수 오류와 해결법

### 오류 1: 정의되지 않은 변수 사용

```python
# ❌ 틀린 예
print(student_name)  # NameError: name 'student_name' is not defined

# ✅ 올바른 예
student_name = "김파이썬"  # 먼저 변수 정의
print(student_name)        # 그 다음 사용
```

### 오류 2: 변수명 오타

```python
# ❌ 틀린 예
student_name = "김파이썬"
print(student_nam)    # NameError (오타!)

# ✅ 올바른 예
student_name = "김파이썬"
print(student_name)   # 정확한 이름 사용
```

### 오류 3: 타입 혼동

```python
# ❌ 틀린 예
age = "20"           # 문자열
next_age = age + 1   # TypeError: can't add int to string

# ✅ 올바른 예
age = 20             # 숫자
next_age = age + 1   # 정상 작동
```

**오늘의 프로그램**: 정보를 기억하고 활용하는 지능형 시스템

### 📦 마법의 상자를 얻은 의미

이제 여러분의 프로그램은:

- **과거를 기억**합니다 (사용자가 입력한 정보 저장)
- **현재를 인식**합니다 (저장된 데이터로 현 상황 파악)
- **미래를 계획**합니다 (목표와 비교하여 다음 행동 결정)

### 🎭 프로그래밍의 3대 기둥 완성

1. ✅ **창조** (Hello World): 무에서 유를 만드는 힘
2. ✅ **소통** (입출력): 사용자와 대화하는 능력
3. ✅ **기억** (변수): 정보를 저장하고 활용하는 지능
