프로그래밍을 시작하는 모든 개발자가 거쳐가는 첫 번째 단계로, 다음과 같은 의미를 가집니다:

## 💻 첫 번째 파이썬 프로그램 작성하기

### 1단계: 새 파일 만들기

1. **VS Code 실행**
2. `Ctrl + N` (새 파일 생성)
3. `Ctrl + S`로 저장
   - 파일명: `hello.py`
   - 위치: 바탕화면이나 문서 폴더의 새 폴더 (예: `python_study`)

> **💡 팁**: `.py` 확장자가 중요합니다. 이것이 파이썬 파일임을 VS Code에게 알려주는 신호입니다.

### 2단계: 코드 작성

다음 코드를 **정확히** 입력하세요:

```python
print("Hello, World!")
```

**주의사항**:

- `print`는 **소문자**로만 작성
- 따옴표는 **영어** 따옴표 사용 (`"` 또는 `'`)
- 괄호와 따옴표를 정확히 짝 맞추기

### 3단계: 프로그램 실행하기

#### 방법 1: VS Code 실행 버튼 (추천)

1. 에디터 우상단의 **▶️ 버튼** 클릭
2. 또는 `Ctrl + F5` 단축키 사용

#### 방법 2: 터미널에서 실행

1. `Ctrl + \`` (백틱)으로 VS Code 내장 터미널 열기
2. 다음 명령어 입력:

```bash
python hello.py
```

### 4단계: 결과 확인

성공하면 다음과 같이 출력됩니다:

```
Hello, World!
```

## 🎉 축하합니다! 첫 번째 프로그램 완성!

비록 한 줄짜리 간단한 프로그램이지만, 이 프로그램은 프로그래밍의 핵심 과정을 모두 포함하고 있습니다:

- **입력**: 소스 코드 (`print("Hello, World!")`)
- **처리**: 파이썬 인터프리터가 코드를 해석
- **출력**: 화면에 텍스트 표시

## 🔍 코드 분석: `print` 함수 이해하기

### `print` 함수란?

```python
print("Hello, World!")
#  ↑        ↑
# 함수    메시지
```

- **`print`**: 화면에 무언가를 출력하는 **함수(function)**
- **`"Hello, World!"`**: 출력할 **문자열(string)**
- **`()`**: 함수에 정보를 전달하는 **괄호**

### 다양한 메시지 출력해보기

```python
print("안녕하세요!")
print("저는 파이썬을 배우고 있어요")
print("Hello, Python!")
print("2024년 파이썬 시작!")
```

### 여러 줄 출력하기

```python
print("첫 번째 줄")
print("두 번째 줄")
print("세 번째 줄")
```

출력 결과:

```
첫 번째 줄
두 번째 줄
세 번째 줄
```

## 🚨 자주 발생하는 오류와 해결법

### 오류 1: `NameError: name 'Print' is not defined`

**문제**: `Print`를 대문자로 작성

```python
Print("Hello, World!")  # ❌ 틀림
```

**해결**: 소문자로 작성

```python
print("Hello, World!")  # ✅ 올바름
```

### 오류 2: `SyntaxError: invalid character`

**문제**: 한글 따옴표 사용

```python
print("Hello, World!")  # ❌ 한글 따옴표
```

**해결**: 영어 따옴표 사용

```python
print("Hello, World!")  # ✅ 영어 따옴표
```

### 오류 3: `SyntaxError: unexpected EOF while parsing`

**문제**: 괄호를 닫지 않음

```python
print("Hello, World!"  # ❌ 닫는 괄호 없음
```

**해결**: 괄호 짝 맞추기

```python
print("Hello, World!")  # ✅ 괄호 완성
```

### 오류 4: `FileNotFoundError`

**문제**: 파일을 저장하지 않고 실행  
**해결**: `Ctrl + S`로 파일 저장 후 실행

## 💡 퀴즈: Hello World 마스터하기

### Q1. 다음 중 올바른 파이썬 코드는?

1. `Print("Hello")`
2. `print('Hello')`
3. `print("Hello")`
4. `PRINT("Hello")`

<details>
<summary>💡 정답 확인</summary>

**정답: 2번과 3번 모두 정답**

파이썬에서는 작은따옴표(`'`)와 큰따옴표(`"`) 모두 사용 가능하지만, `print`는 반드시 소문자여야 합니다.

</details>

### Q2. `print("안녕")`을 실행하면 무엇이 출력될까요?

1. print("안녕")
2. "안녕"
3. 안녕
4. 오류 발생

<details>
<summary>💡 정답 확인</summary>

**정답: 3번**

`print` 함수는 따옴표 안의 내용만 출력하고, 따옴표 자체는 출력하지 않습니다.

</details>

### Q3. 여러 줄을 출력하려면?

1. `print("첫줄 둘째줄")`
2. `print("첫줄") print("둘째줄")`
3. `print("첫줄")` 다음 줄에 `print("둘째줄")`
4. `print(첫줄, 둘째줄)`

<details>
<summary>💡 정답 확인</summary>

**정답: 3번**

각각의 `print` 문은 별도의 줄에 작성해야 하며, 각 `print`는 자동으로 줄바꿈을 추가합니다.

</details>
