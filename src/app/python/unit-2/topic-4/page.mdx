## 🧠 기억에서 인식으로: 프로그램의 마지막 진화

### 위대한 여정의 마지막 단계

- **1단계**: Hello World로 **창조**의 힘을 얻었습니다
- **2단계**: 입출력으로 **소통**의 능력을 익혔습니다
- **3단계**: 변수로 **기억**의 마법을 배웠습니다
- **4단계**: 이제 **인식**의 지혜를 완성할 차례입니다!

### 🔍 인식이란 무엇인가?

지금까지 변수라는 상자에 모든 것을 담았지만, **어떤 종류의 데이터인지 구분하지 못했습니다**:

```python
mystery_box1 = 25        # 이게 나이? 점수? 가격?
mystery_box2 = "25"      # 이건 문자? 숫자?
mystery_box3 = 25.5      # 소수점이 있네?
mystery_box4 = True      # 이건 또 뭐지?
```

### 🌟 데이터 타입: 프로그램의 안목

**데이터 타입(Data Type)**은 프로그램이 **데이터의 종류를 구분하고 적절히 처리하는 능력**입니다:

- 숫자면 → 계산하고
- 문자면 → 연결하고
- 참/거짓이면 → 판단하고

> **인식의 철학**:
> "구분할 줄 아는 것이 진정한 지능이다"
>
> 데이터 타입을 이해한다는 것은 프로그램에게 **세상을 구분해서 바라보는 안목**을
> 선사하는 것입니다! 👁️✨

### 🔢 숫자 타입 변환

```python

int_from_str = int(str_num)
```

## 🔄 PART 4: 타입 변환과 확인 - 자유자재로 변신하기

### 타입 확인하기

```python
# 🔍 타입 탐정
data_list = [42, 3.14, "Hello", True, "123"]

print("🕵️ 타입 탐정 결과")
print("="*30)

for i, data in enumerate(data_list):
    data_type = type(data).__name__
    print(f"{i+1}. {data} → {data_type}")

    # 각 타입별 특징 설명
    if isinstance(data, int):
        print(f"   정수: 계산 가능, 다음 수는 {data + 1}")
    elif isinstance(data, float):
        print(f"   실수: 소수점 {len(str(data).split('.')[1])}자리")
    elif isinstance(data, str):
        print(f"   문자열: {len(data)}글자, 대문자: {data.upper()}")
    elif isinstance(data, bool):
        print(f"   불린: {'참' if data else '거짓'}")
    print()
```

### 프로젝트: 만능 계산기

```python
# 🧮 만능 타입 변환 계산기
print("🔢 만능 계산기")
print("="*25)

def safe_convert_to_number(value):
    """문자열을 숫자로 안전하게 변환"""
    try:
        # 정수로 변환 시도
        if '.' not in value:
            return int(value), 'int'
        else:
            return float(value), 'float'
    except ValueError:
        return None, 'invalid'

while True:
    print("\n💡 두 수를 입력하세요 (종료: 'q')")

    # 첫 번째 수 입력
    input1 = input("첫 번째 수: ")
    if input1.lower() == 'q':
        print("👋 계산기를 종료합니다!")
        break

    # 두 번째 수 입력
    input2 = input("두 번째 수: ")
    if input2.lower() == 'q':
        print("👋 계산기를 종료합니다!")
        break

    # 타입 변환
    num1, type1 = safe_convert_to_number(input1)
    num2, type2 = safe_convert_to_number(input2)

    if num1 is None or num2 is None:
        print("❌ 숫자가 아닌 값이 입력되었습니다!")
        continue

    # 연산 선택
    operation = input("연산 선택 (+, -, *, /, //, %, **): ")

    # 계산 및 결과
    print(f"\n📊 계산 결과")
    print("="*25)
    print(f"입력1: {input1} → {num1} ({type1})")
    print(f"입력2: {input2} → {num2} ({type2})")

    try:
        if operation == '+':
            result = num1 + num2
        elif operation == '-':
            result = num1 - num2
        elif operation == '*':
            result = num1 * num2
        elif operation == '/':
            result = num1 / num2
        elif operation == '//':
            result = num1 // num2
        elif operation == '%':
            result = num1 % num2
        elif operation == '**':
            result = num1 ** num2
        else:
            print("❌ 지원하지 않는 연산입니다!")
            continue

        print(f"연산: {num1} {operation} {num2} = {result}")
        print(f"결과 타입: {type(result).__name__}")

    except ZeroDivisionError:
        print("❌ 0으로 나눌 수 없습니다!")
    except Exception as e:
        print(f"❌ 계산 오류: {e}")
```
