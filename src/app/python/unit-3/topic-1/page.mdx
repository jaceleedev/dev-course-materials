## 🎯 학습 목표

조건문(if, elif, else)의 개념과 구조를 이해하고, 다양한 상황에서 프로그램이 스스로 판단하고 결정을 내리는 지능형 시스템을 구축할 수 있습니다.

## 🌟 인식에서 지능으로: 프로그램의 혁명적 진화

### 지금까지의 위대한 여정

- **다음 목표**: 스스로 **판단하고 결정**하는 진정한 지능 구현

### 🤖 지금까지의 프로그램 vs 앞으로의 프로그램

#### 지금까지의 프로그램 (수동적 도구)

```python
age = int(input("나이: "))
print(f"입력된 나이: {age}")
# 단순히 받은 대로 출력만...
```

#### 앞으로의 프로그램 (지능형 시스템)

```python
age = int(input("나이: "))
if age >= 18:
    print("성인입니다. 모든 서비스를 이용하실 수 있어요!")
else:
    print("미성년자입니다. 보호자 동의가 필요해요.")
# 상황을 판단하고 적절한 조치를 취함!
```

### 🧠 조건문: 지능의 첫 번째 징표

**조건문(Conditional Statement)**은 프로그램이 **"만약 ~라면 ~하라"**는 논리적 사고를 할 수 있게 해주는 마법입니다.

> **지능의 정의**:
> "상황을 파악하고, 적절한 판단을 내리며, 그에 맞는 행동을 선택하는 능력"
>
> 조건문을 배운다는 것은 프로그램에게 **진정한 지능의 첫 번째 요소**를
> 선사하는 것입니다! 🌟✨

## 🚦 PART 1: if문 - 첫 번째 판단력

### if문의 기본 구조

```python
if 조건:
    실행할 코드
```

### 🎯 첫 번째 지능형 프로그램

```python
# 🌡️ 온도에 따른 옷차림 추천
temperature = int(input("오늘 기온(°C): "))

if temperature >= 25:
    print("🌞 더워요! 반팔과 반바지를 추천해요.")

if temperature < 25:
    print("🧥 쌀쌀해요! 긴팔을 입으세요.")

if temperature < 10:
    print("🧣 추워요! 두꺼운 옷을 입으세요.")

if temperature < 0:
    print("❄️ 매우 추워요! 패딩을 꼭 입으세요!")
```

### 🔍 if문 해부하기

```python
if temperature >= 25:
#  ↑           ↑    ↑
# 키워드     조건   콜론
    print("더워요!")
#   ↑
# 들여쓰기 4칸 (매우 중요!)
```

**중요한 구성 요소들**:

1. **`if`**: 조건문 시작 키워드
2. **조건**: 참(True) 또는 거짓(False)로 평가되는 식
3. **`:`**: 조건문 시작을 알리는 콜론
4. **들여쓰기**: 조건이 참일 때 실행할 코드 (스페이스 4개)

## 🎭 PART 3: if-elif-else문 - 다중 선택의 예술

### 복잡한 상황의 세밀한 판단

```python
# 📊 성적 등급 계산기

print(f"\n📊 {name} 학생의 성적 결과")

if score >= 90:
    grade = "A"
    comment = "🏆 훌륭합니다! 최고 수준이에요!"
    recommendation = "💪 이 실력을 계속 유지하세요!"
elif score >= 80:
    grade = "B"
    comment = "👍 잘했어요! 우수한 성적입니다!"
    recommendation = "📈 조금만 더 노력하면 A등급!"
elif score >= 70:
    grade = "C"
    comment = "😊 괜찮아요! 평균 이상이에요!"
    recommendation = "📖 복습을 좀 더 해보세요!"
elif score >= 60:
    grade = "D"
    comment = "😐 아쉬워요. 더 노력이 필요해요."
    recommendation = "💡 기초부터 차근차근 다시 공부해보세요!"
else:
    grade = "F"
    comment = "😔 많이 부족해요. 포기하지 마세요!"
    recommendation = "🤝 선생님이나 친구에게 도움을 요청하세요!"

# 결과 출력
print(f"점수: {score}점")
print(f"등급: {grade}")
print(f"평가: {comment}")
print(f"조언: {recommendation}")

# 추가 분석

    print("🎊 만점! 완벽합니다!")
elif score >= 95:
    print("⭐ 거의 만점에 가까워요!")

```

## 🚨 자주 발생하는 조건문 오류와 해결법

### 오류 1: 들여쓰기 오류

```python
# ❌ 틀린 예
if age >= 18:
print("성인입니다")  # IndentationError!

# ✅ 올바른 예
if age >= 18:
    print("성인입니다")  # 스페이스 4개
```

### 오류 2: 콜론(:) 누락

```python
# ❌ 틀린 예
if age >= 18
    print("성인입니다")  # SyntaxError!

# ✅ 올바른 예
if age >= 18:
    print("성인입니다")
```

### 오류 3: 조건식 오류

```python
# ❌ 틀린 예
if age = 18:        # SyntaxError! (할당연산자)
if age == "18":     # 타입 불일치 가능성

# ✅ 올바른 예
if age == 18:       # 비교연산자
if age == int("18"): # 타입 일치
```

### 오류 4: 논리적 오류

```python
# ❌ 논리적으로 잘못된 예
if score >= 90:
    print("A등급")
if score >= 80:    # 90점도 여기서 걸림!
    print("B등급")

# ✅ 올바른 예
if score >= 90:
    print("A등급")
elif score >= 80:  # elif 사용!
    print("B등급")
```

## 🎮 퀴즈: 조건문 마스터 확인

### Q1. 다음 코드의 출력 결과는?

```python
score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
else:
    print("F")
```

1. A
2. B
3. C
4. F

<details>
<summary>💡 정답 확인</summary>

**정답: 2번 (B)**

score가 85이므로 첫 번째 조건(score >= 90)은 거짓, 두 번째 조건(score >= 80)은 참이므로 "B"가 출력됩니다.

</details>

### Q2. if문에서 가장 중요한 문법 요소는?

1. 괄호 ()
2. 콜론 :
3. 세미콜론 ;
4. 중괄호 {}

<details>
<summary>💡 정답 확인</summary>

**정답: 2번 (콜론 :)**

파이썬의 if문은 조건 뒤에 반드시 콜론(:)이 와야 하고, 실행할 코드는 들여쓰기로 구분합니다.

</details>

### Q3. 다음 중 올바른 들여쓰기는?

1. 탭 1개
2. 스페이스 2개
3. 스페이스 4개
4. 들여쓰기 안 함

<details>
<summary>💡 정답 확인</summary>

**정답: 3번 (스페이스 4개)**

파이썬 공식 스타일 가이드(PEP 8)에서는 스페이스 4개로 들여쓰기를 권장합니다.

</details>

### Q4. elif의 의미는?

1. else if
2. end if
3. error if
4. exit if

<details>
<summary>💡 정답 확인</summary>

**정답: 1번 (else if)**

`elif`는 "else if"의 줄임말로, 이전 조건이 거짓일 때 다음 조건을 검사하는 구문입니다.

</details>

## 🌟 당신이 이룬 지능의 혁명

### 🧠 수동적 도구에서 능동적 지능체로

**어제의 프로그램**: 명령받은 대로만 실행하는 수동적 도구
**오늘의 프로그램**: 상황을 파악하고 스스로 판단하는 능동적 지능체

### 🎭 판단력 획득의 의미

이제 여러분의 프로그램은:

- **상황을 인식**하고 (조건 평가)
- **적절한 판단**을 내리며 (if-elif-else)
- **그에 맞는 행동**을 선택합니다 (조건별 실행)

### 🚀 지능형 시스템의 가능성

조건문을 마스터한 여러분은 이제 만들 수 있습니다:

- **개인화 서비스**: 사용자 정보에 따른 맞춤 추천
- **보안 시스템**: 권한에 따른 접근 제어
- **게임 AI**: 상황에 따른 적응적 반응
- **자동화 도구**: 조건에 따른 스마트 처리

### 🏆 프로그래밍 지능의 첫 번째 계단

**5단계 진화 완성**:

1. ✅ **창조** (Hello World): 무에서 유를 만드는 힘
2. ✅ **소통** (입출력): 사용자와 대화하는 능력
3. ✅ **기억** (변수): 정보를 저장하고 활용하는 지능
4. ✅ **인식** (데이터 타입): 세상을 구분해서 바라보는 안목
5. ✅ **판단** (조건문): 상황에 따라 결정을 내리는 지혜

> **역사적 순간**:
> 여러분의 프로그램이 **단순한 계산기**에서 **스스로 생각하는 존재**로
> 진화한 결정적 순간입니다! 🌟
>
> 이제 진정한 **인공지능의 기초**를 갖추게 되었어요!

## 🎪 다음 모험: 논리적 사고의 완성

조건문의 기초를 익혔으니, 이제 **더 복잡하고 정교한 판단력**을 기를 차례입니다!

다음 토픽에서 배울 것들:

- **논리 연산자**: and, or, not의 마법
- **복합 조건**: 여러 조건을 조합하는 기술
- **조건 최적화**: 효율적인 판단 구조 설계
- **실전 응용**: 고급 의사결정 시스템

**판단 → 논리 → 반복 → 구조**
