# Topic 1: 프로그래밍이라는 새로운 언어 🗣️

## 🌍 언어란 무엇일까요?

> "언어는 생각을 표현하고 소통하는 도구입니다"

우리는 매일 한국어로 생각하고, 영어로 세계와 소통하며, 때로는 손짓과 몸짓으로도 의사를 전달합니다.

**프로그래밍 언어도 마찬가지입니다.** 다만 소통의 상대가 사람이 아닌 **컴퓨터**일 뿐이에요.

![프로그래밍 코드 이미지](/images/python-code-screen.webp)

## 🤝 사람 vs 컴퓨터, 소통 방식의 차이

### 사람과의 소통

- **한국어**: "커피 한 잔 주세요"
- **영어**: "Can I have a cup of coffee?"
- **몸짓**: 컵 모양을 손으로 만들며 마시는 시늉

### 컴퓨터와의 소통 (프로그래밍)

- **Python**: `print("커피 한 잔 주세요")`
- **Java**: `System.out.println("Can I have a cup of coffee?");`
- **JavaScript**: `console.log("커피 한 잔 주세요");`

## 📚 외국어 학습 vs 프로그래밍 언어 학습

| --------------------- | ----------------------- |

## 🔍 실제 코드 예시: 다양한 언어로 같은 내용 표현하기

"1부터 10까지 숫자를 출력하라"는 같은 요구사항을 다양한 언어로 표현해보겠습니다.

### Python (파이썬)

```python
for i in range(1, 11):
  print(i)
```

### JavaScript (자바스크립트)

```javascript
for (let i = 1; i < 11; i++) {
  console.log(i);
}
```

### Java (자바)

```java
for (int i = 1; i < 11; i++) {
  System.out.println(i);
}
```

### C (C언어)

```c
for (int i = 1; i < 11; i++) {
  printf("%d\n", i);
}
```

**같은 의미, 다른 표현!** 마치 "안녕하세요"를 영어로는 "Hello", 일본어로는 "こんにちは"라고 하는 것과 같아요.

## 🎯 외국어와 다른 프로그래밍 언어의 특징

### 1. 정확성이 생명

영어는 문법을 좀 틀려도 상대방이 이해합니다. 프로그래밍 언어에서는 **문법 오류 하나로도 프로그램이 작동하지 않습니다**.

```python
# 올바른 문법
print("Hello, World!")

# 잘못된 문법 (대문자 P)
Print("Hello, World!")  # NameError 발생!
```

### 2. 논리적 순서

요리할 때 재료를 먼저 준비하고 요리하듯이, 프로그래밍도 **필요한 것을 먼저 정의하고 나서 사용**해야 합니다.

```python
# 올바른 순서 (재료 준비 → 요리)
name = "홍길동"           # 1. 변수 준비
print(f"안녕, {name}!")   # 2. 변수 사용

# 잘못된 순서 (요리 → 재료 준비)
print(f"안녕, {name}!")   # NameError! name이 아직 없어요
name = "홍길동"
```
